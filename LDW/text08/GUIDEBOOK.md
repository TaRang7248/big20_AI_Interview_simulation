# 면접 프로그램 가이드북 (GUIDEBOOK)

이 가이드북은 AI 면접 시뮬레이션 프로그램의 사용법과 새로운 기능을 설명합니다.

## 새로운 기능: 음성 질문 안내 (TTS)
면접 질문이 화면에 표시될 때, AI 면접관의 목소리로 질문을 읽어줍니다. 

**작동 방식:**
1. 질문이 화면에 나타납니다.
2. 동시에 음성으로 질문을 읽어줍니다.
3. **음성 안내가 끝난 후** 타이머가 시작됩니다.
4. 타이머가 시작되면 답변을 시작하세요.

## 프로그램 시작 방법
1. 터미널에서 `python start_app.py`를 실행합니다.
2. 브라우저가 자동으로 열립니다. (`http://127.0.0.1:8000`)

## 면접 진행 순서
1. **메인 화면**: '면접 시작하기' 버튼을 클릭합니다.
2. **면접 진행**:
   - 웹캠 사용 권한을 허용해주세요.
   - 질문을 듣고 답변을 준비합니다.
   - '녹화 시작' 버튼을 누르거나 타이머가 종료되면 자동으로 답변 시간이 흐릅니다. (수동 시작 필요)
   - 답변을 마치면 '녹화 종료'를 누릅니다.
3. **피드백**: 모든 질문이 끝나면 AI의 피드백을 확인합니다.

## 주의 사항
- 브라우저의 소리를 켜주세요.
- TTS 기능은 브라우저의 `SpeechSynthesis` API를 사용하므로, 최신 브라우저(Chrome, Edge 등) 사용을 권장합니다.
- 음성이 나오지 않는 경우 브라우저 설정을 확인해주세요.
