# AI 면접 프로그램 가이드북

이 가이드북은 AI 면접 시뮬레이션 프로그램의 사용법과 주요 기능을 설명합니다.
본 프로그램은 **100% Python 언어**로 동작하도록 수정되었습니다. (Frontend: PyScript, Backend: FastAPI)

## 1. 프로그램 개요
이 프로그램은 모의 면접을 통해 답변 연습을 하고, AI로부터 피드백을 받을 수 있는 웹 애플리케이션입니다. 브라우저에서 Python 코드를 직접 실행하여 면접 로직을 처리합니다.

## 2. 주요 기능 및 변경 사항

### 2.1 인터뷰 진행 (Python 기반)
*   **PyScript 적용**: 기존 자바스크립트 로직이 모두 Python(`static/py/*.py`)으로 교체되었습니다.
*   **질문 제시**: 화면에 면접 질문이 텍스트와 음성으로 제시됩니다.
*   **타이머**: 각 질문에는 제한 시간이 주어지며 Python 타이머 로직이 작동합니다.
*   **답변 녹음**:
    *   `답변 시작` 버튼을 눌러 답변을 녹음합니다. (브라우저의 MediaRecorder API를 Python에서 제어)
    *   `답변 종료` 버튼을 누르면 녹음이 완료되고 답변이 제출됩니다.

### 2.2 시간 초과 처리
*   **녹음 중인 경우**: 시간이 초과되면 즉시 녹음을 중단하고 자동 제출됩니다.
*   **답변을 하지 않은 경우**: "대답하지 않음"으로 처리되고 다음 질문으로 넘어갑니다.

### 2.3 시스템 아키텍처 그리기
*   화면 우측의 캔버스를 통해 아키텍처를 그릴 수 있습니다.
*   이 기능 역시 Python 코드로 마우스 이벤트를 처리하여 동작합니다.

## 3. 기술적 변경 사항 (개발자용)
*   **JavaScript 제거**: `canvas.js`, `interview.js`, `main.js` 파일이 삭제되었습니다.
*   **Python 스크립트 추가**:
    *   `static/py/login_handler.py`: 로그인 및 면접 시작 처리
    *   `static/py/interview_flow.py`: 면접 진행 및 미디어 처리
    *   `static/py/canvas_handler.py`: 캔버스 드로잉 로직
*   **HTML**: 기존 `<script>` 태그 대신 `<script type="py">`를 사용하여 Python 파일을 로드합니다.

## 4. 문제 해결
*   **로딩 속도**: PyScript 초기 구동 시 Python 런타임을 다운로드하므로 첫 로딩이 조금 걸릴 수 있습니다.
*   **웹캠/마이크 오류**: 브라우저 권한을 확인해주세요.
