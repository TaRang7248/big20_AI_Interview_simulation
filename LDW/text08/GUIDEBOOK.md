# AI 면접 프로그램 가이드북 (GUIDEBOOK)

본 프로그램은 백엔드 서버 없이 브라우저에서 즉시 실행 가능한 **웹 기반 AI 면접 시뮬레이션**입니다. 설계서(`AI 면접 프로그램 UI 설계 초안.md`)를 기반으로 핵심 UI와 플로우를 구현하였습니다.

## 1. 파일 구조 (File Structure)
*   **index.html**: 애플리케이션의 진입점. SPA(Single Page Application) 구조로 모든 화면 섹션을 포함합니다.
*   **styles.css**: 전체적인 디자인 스타일링 (레이아웃, 색상, 컴포넌트)을 담당합니다.
*   **app.js**: 핵심 로직. 라우팅, Mock 데이터 관리, 인증, 면접 진행, 결과 계산 등을 처리합니다.

## 2. 실행 방법 (How to Run)
1.  이 폴더(`c:\big20\big20_AI_Interview_simulation\LDW\text08`)를 엽니다.
2.  `index.html` 파일을 더블 클릭하여 웹 브라우저(Chrome, Edge 등)에서 실행합니다.
    *   *별도의 서버 설치가 필요 없습니다.*

## 3. 사용 시나리오 (Usage Scenario)

### 3.1 회원가입 및 로그인
*   **테스트 계정**:
    *   **면접자**: ID: `test` / PW: `1234`
    *   **관리자**: ID: `admin` / PW: `admin`
*   **회원가입**: 로그인 화면 하단 '회원가입' 링크를 통해 신규 사용자 등록이 가능합니다 (브라우저 새로고침 전까지 유지).

### 3.2 면접 진행 (면접자)
1.  `test` 계정으로 로그인합니다.
2.  대시보드에서 [지원하기] 버튼을 클릭합니다.
3.  [장치 테스트 실행] 버튼을 눌러 카메라/마이크 점검(시뮬레이션)을 완료합니다.
4.  [면접 시작]을 누르면 면접 룸으로 입장합니다.
5.  **진행 단계**:
    *   AI 질문 등장 -> 타이머 작동 -> 답변 입력 -> [답변 완료] 클릭.
    *   Phase 1 (자기소개) -> Phase 2 (역량) -> Phase 3 (코딩/화이트보드) 순으로 진행됩니다.
6.  모든 질문이 끝나면 결과 페이지로 이동하며, 영역별 점수와 합불 예측 결과를 확인할 수 있습니다.

### 3.3 관리자 기능
1.  `admin` 계정으로 로그인합니다.
2.  **공고 관리**: [공고 등록] 버튼을 통해 새로운 채용 공고를 추가할 수 있습니다.
3.  **지원자 현황**: '지원자 현황' 메뉴에서 면접을 완료한 지원자들의 기록과 점수를 조회할 수 있습니다.

## 4. 참고 사항 (Note)
*   본 프로그램은 **Mock Data**를 사용하므로, 브라우저를 새로고침하면 회원가입 정보나 면접 기록이 초기화됩니다.
*   카메라와 마이크는 실제 장치를 사용하지 않고 시뮬레이션(Mock) 처리되었습니다.
