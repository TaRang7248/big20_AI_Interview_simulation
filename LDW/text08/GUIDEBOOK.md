# AI 면접 시뮬레이션 프로그램 가이드북

본 문서는 AI 면접 시뮬레이션 프로그램의 사용법, 주요 기능, 기술 스택 및 구조에 대해 설명합니다.

## 1. 프로그램 개요
이 프로그램은 실제 면접 상황을 시뮬레이션하여 지원자의 답변 내용과 태도를 AI로 분석하고 실시간 피드백을 제공하는 시스템입니다.

### 주요 특징
- **실전 같은 면접 진행**: 자기소개, 인성 질문, 직무 지식 질문 등 총 10단계의 체계적인 면접 시나리오를 제공합니다.
- **실시간 음성 인식 (STT)**: 지원자의 음성을 실시간으로 텍스트로 변환하여 분석합니다.
- **영상 분석 (Non-verbal Analysis)**: 웹캠을 통해 지원자의 표정과 움직임을 분석하여 자신감과 태도 점수를 산출합니다.
- **AI 기반 평가**: 답변 내용의 적절성, 논리성 등을 LLM(Large Language Model)이 평가합니다.
- **종합 피드백**: 면접 종료 후 답변 점수와 영상 분석 점수를 종합하여 합격 여부를 판정합니다.

---

## 2. 사용 방법

### 시작하기
1. **로그인/설정**: 지원자 이름과 지원 직무를 입력하고 면접을 시작합니다.
2. **면접 진행**:
    - 화면에 질문이 표시되고 음성으로 읽어줍니다.
    - **[답변 시작]** 버튼을 누르고 답변을 말합니다.
    - 답변이 끝나면 **[답변 종료]** 버튼을 누릅니다. (시간 초과 시 자동 종료)
    - 답변은 자동으로 분석되며 다음 질문으로 넘어갑니다.
3. **아키텍처 캔버스**: 직무 질문 답변 시, 필요하다면 우측 캔버스에 그림을 그려 설명을 보충할 수 있습니다.
4. **결과 확인**: 모든 질문이 끝나면 피드백 페이지로 이동하여 상세 점수와 합격 여부를 확인합니다.

### 합격 기준
다음 **두 가지 조건을 모두 만족**해야 합격입니다.
1. **답변 평균 점수**: 70점 이상
2. **영상 분석 점수 (자신감/태도)**: 70점 이상

## 3. 기술 스택 (Tech Stack)

### Backend
- **Language**: Python 3.10+
- **Framework**: FastAPI (비동기 웹 프레임워크)
- **Database**: 
    - SQLite (면접 기록 및 로깅)
    - PostgreSQL + pgvector (RAG 및 벡터 검색용)
- **AI/ML**:
    - `OpenAI API` / `LangChain`: 답변 평가 및 질문 생성
    - `OpenCV`: 실시간 영상 분석 (얼굴 인식, 움직임 분석)
    - `Google Speech-to-Text` (또는 로컬 모델): 음성 인식

### Frontend
- **HTML/CSS/JS**: 직관적인 사용자 인터페이스
- **PyScript**: 브라우저 내 Python 실행 (실시간 오디오 처리 등 일부 기능)

## 4. 문제 해결 (Troubleshooting)

- **화면이 넘어가지 않음**: 브라우저를 새로고침하거나, 웹캠 권한이 허용되어 있는지 확인하세요.
- **피드백 결과가 나오지 않음**: 서버가 재시작되었거나 세션이 만료되었을 수 있습니다. 처음부터 다시 시작해주세요. (현재 버전은 인메모리 세션을 사용합니다.)
- **음성 인식이 안 됨**: 마이크 연결 상태를 확인하고, 조용한 환경에서 진행해 주세요.
- **점수가 낮음**: 목소리를 크게 하고, 카메라를 정면으로 응시하며 움직임을 최소화해 보세요.
