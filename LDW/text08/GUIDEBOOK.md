# 📘 AI Interview Simulation - GUIDEBOOK

## 소개 (Introduction)
이 프로그램은 **GPT-4o** 기반의 AI 면접 시뮬레이터입니다.
지원자가 선택한 직무(Job Title)에 맞춰, 20년차 베테랑 전문가 페르소나를 가진 AI 면접관이 실제 면접과 유사한 경험을 제공합니다.

면접관은 다음의 단계를 통해 지원자를 심층적으로 평가합니다:
1.  **초기 설정**: 직무별 맞춤형 질문 생성
2.  **실시간 상호작용**: 음성 답변 인식(STT) 및 실시간 점수 피드백
3.  **심층 검증**: 답변에 따른 꼬리 질문(Follow-up) 생성
4.  **최종 평가**: 전체 면접 결과 및 합격 여부 판단

---

## 🎯 면접 진행 단계 (Interview Process)
면접은 총 **10단계**의 메인 질문으로 구성되며, 상황에 따라 **꼬리 질문(Follow-up)**이 추가될 수 있습니다.

| 단계 | 유형 | 설명 |
| :-- | :-- | :-- |
| **1단계** | **자기소개** | 지원자의 기본 정보와 강점을 파악하기 위한 자기소개 시간입니다. (LLM 생성) |
| **2~4단계** | **인성/역량** | 지원자의 가치관, 협업 능력, 문제 해결 방식 등 인성 면접이 진행됩니다. |
| **5~9단계** | **직무 지식** | `interview.db` 및 RAG(검색 증강 생성)를 활용하여 직무와 관련된 심도 있는 기술 면접이 진행됩니다. |
| **10단계** | **마무리** | 면접을 정리하며 마지막으로 하고 싶은 말이나 질문을 받습니다. |

> **💡 꼬리 질문 (Tail Questions)**
> 지원자의 답변이 불충분하거나 더 깊이 파고들 필요가 있을 경우, AI 면접관이 추가 질문을 던집니다. 이는 메인 단계 카운트에 포함되지 않으며, 지원자의 답변을 명확히 하기 위함입니다.

## 🚀 실행 방법 (How to Run)

### 1. 환경 설정 (Prerequisites)
- **Python 3.9+** 설치 필요
- `.env` 파일에 `OPENAI_API_KEY` 설정 필수

### 2. 설치 (Installation)
필요한 패키지를 설치합니다.
```bash
pip install -r requirements.txt
```

### 3. 서버 실행 (Run Server)
서버를 실행합니다.
```bash
python start_app.py
```
서버가 시작되면 브라우저에서 `http://127.0.0.1:8000` 또는 `http://127.0.0.1:8000/interview.html` 로 접속하세요.

## 📝 주요 기능 설명 (Key Features)

### 🎙️ 음성 답변 (Voice Answer)
- 사용자는 마이크를 통해 질문에 답변할 수 있습니다.
- 답변이 끝나면 자동으로 **STT(Speech-to-Text)** 변환이 이루어집니다.
- 화면에 **인식된 텍스트**와 해당 답변에 대한 **AI 점수**가 실시간으로 표시됩니다.

### 🎨 아키텍처 캔버스 (Architecture Canvas)
- 기술 면접 시, 말로 설명하기 어려운 시스템 구조나 아키텍처를 그릴 수 있는 캔버스를 제공합니다.
- (현재 버전에서는 그리기 기능만 제공되며, AI 분석에는 포함되지 않을 수 있습니다.)

### 📊 결과 리포트 (Feedback)
- 모든 면접 단계가 종료되면 `feedback.html` 페이지로 이동합니다.
- 전체 답변에 대한 평균 점수와 합격/불합격 판정이 표시됩니다.

## 🛠️ 문제 해결 (Troubleshooting)
- **음성 인식이 안 될 때**: 브라우저 주소창 옆의 '자물쇠' 아이콘을 클릭하여 마이크 권한이 허용되어 있는지 확인하세요.
- **점수가 undefined로 나올 때**: 최신 버전으로 업데이트되었는지 확인하세요. (API `/submit` 응답 필드 수정됨)
- **(Hearing) Audio transcription unavailable 오류**: 백엔드 STT 함수 연동이 수정되었습니다. 서버를 재시작해주세요.
