# 면접 프로그램 가이드북 (GUIDEBOOK)

이 문서는 AI 면접 시뮬레이션 프로그램의 주요 기능과 개선된 기술적 특징을 설명합니다.

## 1. 주요 기능 및 기술적 특징

### 🔍 음성 인식 (STT) 안정화 및 고도화
OpenAI Whisper 모델을 기반으로 하며, 다음과 같은 기술을 적용하여 인식률을 높였습니다.

- **용어 사전 (Custom Vocabulary)**:
    - IT/개발자 직무 면접에 자주 사용되는 전문 용어(예: Kubernetes, Docker, React, Python 등)를 사전 학습된 프롬프트로 제공하여, 영문/한글 혼용 시에도 정확하게 인식합니다.
- **문맥 파악 (Contextual Bias)**:
    - 이전 질문 내용을 음성 인식 엔진에 '문맥'으로 함께 전달합니다. 이를 통해 동음이의어나 모호한 발음을 현재 대화 흐름에 맞게 더 정확하게 변환합니다.

### 📝 텍스트 정규화 (ITN) 및 자연어 처리 (NLP)
음성 인식 결과물(Raw Transcript)을 그대로 사용하지 않고, LLM(GPT-4o)을 활용한 2차 정제 과정을 거칩니다.

- **자동 교정 (Refinement)**:
    - 불필요한 추임새("어...", "음...", "그게...")를 제거합니다.
    - 문법적 오류와 오탈자를 수정하여 문장을 매끄럽게 다듬습니다.
    - 숫자와 단위를 표준 표기법으로 변환합니다 (예: "만오천원" -> "15,000원").
- **전문 용어 대소문자 보정**:
    - "java string" -> "Java String"과 같이 기술 용어의 대소문자를 올바르게 교정합니다.

### 🤖 심층 면접관 페르소나 (LLM)
- 면접관 AI는 단순한 질문 생성기가 아닌, 지원자의 답변을 분석하고 꼬리 질문을 던지는 '전문가' 페르소나를 가집니다.
- 답변의 논리성, 직무 적합성, 구체성을 평가하여 실시간 피드백을 제공합니다.

## 2. 사용 방법

1. **면접 시작**:
    - `start_app.py`를 실행하여 서버를 켭니다.
    - 웹 브라우저에서 `localhost:8000`으로 접속합니다.
    - 이름과 지원 직무를 입력하고 면접을 시작합니다.

2. **답변 진행**:
    - 질문이 나오면 [답변 시작] 버튼을 누르고 음성으로 답변합니다.
    - 답변이 끝나면 [답변 종료] 버튼을 누릅니다. (또는 시간이 다 되면 자동 제출됩니다.)

3. **결과 확인**:
    - 답변이 제출되면 AI가 음성을 텍스트로 변환(STT)하고, 내용을 정제(NLP)한 뒤 평가합니다.
    - 화면에 변환된 텍스트와 AI 면접관의 피드백, 점수가 표시됩니다.

## 3. 문제 해결 (Troubleshooting)

- **음성 인식이 잘 안 될 때**: 마이크 권한이 허용되어 있는지 확인하고, 가능한 조용한 환경에서 답변해 주세요.
- **웹캠 오류**: 브라우저의 카메라 접근 권한을 허용해 주세요.
