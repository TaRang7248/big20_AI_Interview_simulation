# AI 면접 시뮬레이션 프로그램 가이드북

이 프로그램은 OpenAI의 GPT-4o 및 Whisper 모델을 활용하여 실제와 유사한 면접 경험을 제공하는 AI 면접 시뮬레이션 도구입니다.

## 1. 시작하기 전에

### 환경 설정
- **Python**: Python 3.8 이상의 버전이 필요합니다.
- **OpenAI API Key**: `.env` 파일에 유효한 `OPENAI_API_KEY`가 설정되어 있어야 합니다.
- **마이크 권한**: 웹 브라우저에서 마이크 접근 권한을 허용해야 음성 답변이 가능합니다.

### 실행 방법
1. 터미널에서 프로그램 루트 폴더로 이동합니다.
2. 다음 명령어를 입력하여 서버를 실행합니다:
   ```bash
   python main.py
   ```
3. 서버가 시작되면 자동으로 기본 웹 브라우저가 열립니다 (`http://127.0.0.1:8000`).

## 2. 주요 기능 및 인터페이스

### 메인 화면 (면접 시작)
- **이름**: 면접자의 이름을 입력합니다.
- **지원 직무**: 지원하고자 하는 직무(예: 백엔드 개발자, 데이터 분석가 등)를 입력합니다.
- **면접 시작하기 버튼**: 클릭 시 첫 번째 질문(자기소개)이 항상 생성되며 면접이 시작됩니다.

### 면접 화면
- **진행 표시줄**: 총 10개의 질문 중 현재 진행 상황을 보여줍니다.
- **타이머**: 각 질문당 90초의 답변 시간이 주어집니다.
- **질문 박스**: AI 면접관이 던지는 질문이 표시됩니다.
- **답변 영역**: '답변 시작' 버튼을 누르고 말을 하면 실시간으로 텍스트로 변환되어 표시됩니다. '답변 완료'를 누르면 AI가 답변을 분석합니다.

### 결과 화면
- **합격/불합격 판정**: 전체 면접 답변 내용을 종합하여 최종 결과를 보여줍니다.
- **상세 피드백**: 각 질문별 답변에 대한 점수와 구체적인 개선 제안을 확인할 수 있습니다.

## 3. 면접 팁

1. **명확한 발음**: Whisper 모델이 답변을 정확히 인식할 수 있도록 조용한 환경에서 명확하게 말씀해 주세요.
2. **구체적인 답변**: AI 면접관은 답변의 전문성과 구체성을 평가합니다. 자신의 경험을 구체적인 수치나 사례와 함께 설명하면 좋은 점수를 받을 수 있습니다.
3. **시간 관리**: 90초 이내에 핵심 내용을 전달하는 연습을 하세요. 시간이 초과되면 자동으로 답변이 종료됩니다.
