# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK.md)

## 1. 개요 (Overview)
본 프로젝트는 **웹 기반 AI 면접 시뮬레이션** 프로그램입니다.  
사용자는 지원자로서 이력서를 등록하고, 채용 공고에 맞춘 AI 면접관의 질문에 음성으로 답변하며 실제 면접과 유사한 경험을 할 수 있습니다.  
면접 종료 후에는 기술 역량, 문제 해결 능력, 의사소통, 태도 등 다양한 영역에서 평가된 결과를 확인할 수 있습니다.

## 2. 개발 환경 (Environment)
본 프로그램은 다음 환경에서 개발 및 구동됩니다.
- **OS**: Windows / Mac / Linux (Cross-platform)
- **Language**: 
  - Backend: Python 3.x
  - Frontend: HTML5, CSS3, Vanilla JavaScript
- **Database**: PostgreSQL (필수), SQLite (개발용 Mock 지원 가능)
- **Browser**: Chrome, Edge 등 최신 브라우저 (Web Speech API 지원 필수)

## 3. 프로그램 실행 방법 (Execution)

### 3.1 사전 준비
1. Python 및 PostgreSQL이 설치되어 있어야 합니다.
2. 프로젝트 폴더 내 `.env` 파일에 데이터베이스 접속 정보(`DB_PASSWORD` 등)가 설정되어 있어야 합니다.

### 3.2 서버 실행
터미널(CMD/Shell)을 열고 프로젝트 경로로 이동한 뒤 아래 명령어를 실행합니다.

```sh
python server.py
# 또는
python3 server.py
```

### 3.3 접속
서버가 정상적으로 실행되면 웹 브라우저가 자동으로 실행되며 다음 주소로 접속됩니다.
- URL: `http://localhost:5000`

## 4. 사용 라이브러리 (Libraries)

### Backend (Python)
- **Flask**: 웹 서버 프레임워크
- **psycopg2**: PostgreSQL 데이터베이스 연동
- **pypdf**: PDF 이력서 텍스트 추출
- **python-dotenv**: 환경 변수 관리
- **webbrowser**: 서버 시작 시 브라우저 자동 오픈

### Frontend (JS)
- **Web Speech API**: (내장 API) 음성 인식(STT) 및 음성 합성(TTS)
- **MediaDevices API**: (내장 API) 카메라 및 마이크 장치 제어

## 5. 주요 기능 사용법 (Usage)

### 5.1 회원 가입 및 로그인
1. 메인 화면에서 회원가입 버튼을 클릭하여 아이디, 비밀번호, 이름 등을 입력하여 가입합니다.
2. '관리자' 유형으로 가입하면 공고를 등록/수정/삭제할 수 있는 권한을 가집니다.

### 5.2 채용 공고 관리 (관리자)
1. 관리자 계정으로 로그인합니다.
2. '공고 등록' 버튼을 눌러 직무, 제목, 내용, 마감일을 입력하고 등록합니다.

### 5.3 면접 진행 (지원자)
1. 지원자 계정으로 로그인합니다.
2. 지원 가능한 공고 목록에서 '확인하기'를 누른 후 상세 내용을 확인하고 '지원하기'를 클릭합니다.
3. **이력서 업로드**: PDF 형식의 이력서를 업로드합니다.
4. **환경 테스트**: 카메라와 마이크가 정상 작동하는지 테스트합니다.
5. **면접 시작**:
   - AI 면접관이 직무와 이력서를 기반으로 생성된 질문을 **음성으로 읽어줍니다.**
   - 질문이 끝난 후 타이머(60초)가 작동하며 마이크가 활성화됩니다.
   - **음성으로 답변**하면 실시간으로 텍스트로 변환되어 화면에 표시됩니다.
   - '답변 완료' 버튼을 누르거나 시간이 종료되면 다음 질문으로 넘어갑니다.

### 5.4 결과 확인
1. 모든 질문(기본 5개)이 끝나면 면접이 종료됩니다.
2. AI 모델(모의 로직)이 평가한 영역별 점수와 합격 예측 결과를 그래프와 함께 확인할 수 있습니다.
