# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK)

## 1. 개요 (Overview)
본 프로젝트는 **웹 기반 AI 면접 시뮬레이션** 프로그램입니다.  
사용자는 가상의 AI 면접관과 함께 모의 면접을 진행하며, 음성 인식(STT)과 음성 합성(TTS) 기술을 통해 실전과 유사한 면접 환경을 체험할 수 있습니다.  
면접 종료 후에는 기술 역량, 문제 해결 능력, 의사 소통 능력, 태도 등을 분석한 결과를 제공합니다.

---

## 2. 개발 및 실행 환경 (Environment)
- **OS**: Windows (권장)
- **Language**: Python 3.x, JavaScript (ES6+)
- **Browser**: Chrome, Edge 등 모던 브라우저 (카메라/마이크 권한 필요)
- **Database**: SQLite3 (내장)

---

## 3. 프로그램 실행 방법 (How to Run)
1. 소스 코드가 위치한 디렉토리(`C:\big20\big20_AI_Interview_simulation\LDW\text09`)로 이동합니다.
2. 터미널(CMD 또는 PowerShell)에서 아래 명령어를 입력하여 서버를 실행합니다.
   ```bash
   python server.py
   ```
3. 서버가 정상적으로 실행되면 브라우저가 자동으로 열리며 `http://localhost:5000`으로 접속됩니다.
4. 만약 자동으로 열리지 않을 경우, 브라우저 주소창에 직접 입력하여 접속하세요.

---

## 4. 사용 라이브러리 (Libraries)
### Backend (Python)
- **Flask**: 웹 서버 프레임워크
- **sqlite3**: 내장 데이터베이스 관리
- **webbrowser**: 브라우저 자동 실행 제어

### Frontend (User Interface)
- **Vanilla JS**: 프레임워크 없는 순수 자바스크립트 구현 (SPA 형태)
- **CSS3**: 스타일링 및 레이아웃
- **HTML5**: 웹 페이지 구조 (MediaDevices API 활용)

---

## 5. 주요 기능 사용법 (Key Features)

### 5.1 회원가입 및 로그인
- **회원가입**: ID, 비밀번호, 이름, 생년월일, 성별, 이메일, 주소, 전화번호를 입력하여 계정을 생성합니다.
- **로그인**: 생성한 계정으로 로그인합니다.

### 5.2 내 정보 관리
- **본인 확인**: 정보 수정을 위해 비밀번호 재확인을 거칩니다.
- **정보 수정**: ID, 이름, 생년월일, 성별은 **수정 불가**하며, 이메일, 주소, 전화번호는 **수정 가능**합니다.

### 5.3 면접 공고 지원 및 환경 설정
- **공고 목록**: 지원 가능한 공고를 확인하고 '지원하기' 버튼을 클릭합니다.
- **환경 테스트**: 카메라와 마이크가 정상적으로 작동하는지 테스트합니다. (권한 허용 필요)

### 5.4 AI 실전 면접
- **프로세스**: 자기소개 -> 역량 검증 -> 기술 문제(코딩) 단계로 진행됩니다.
- **상호작용**: AI 면접관이 질문을 읽어주며(TTS), 사용자는 마이크를 통해 음성으로 답변합니다(STT).

### 5.5 결과 분석
- 면접 종료 후 각 영역별 점수와 합격 여부 예측 결과를 그래프와 함께 확인할 수 있습니다.
