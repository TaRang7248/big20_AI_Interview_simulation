# AI 면접 시뮬레이션 프로그램 가이드북

본 가이드북은 웹 기반 AI 면접 시뮬레이션 시스템의 개요, 설치 환경, 실행 방법 및 주요 기능에 대해 설명합니다.

---

## 1. 개요
이 프로그램은 지원자가 업로드한 이력서를 바탕으로 AI 면접관이 직무 맞춤형 질문을 던지고, 지원자의 답변을 실시간으로 분석하여 평가 결과를 제공하는 면접 시뮬레이션 시스템입니다. 
- **프론트엔드**: HTML, Vanilla CSS, Javascript
- **백엔드**: FastAPI (Python)
- **데이터베이스**: PostgreSQL

## 2. 환경 설정
시스템을 실행하기 위해서는 아래와 같은 환경이 필요합니다.

### 시스템 요구사항
- **Python**: 3.9 이상
- **PostgreSQL**: 13 이상
- **오디오 녹음 장치**: 마이크 및 스피커
- **영상 입력 장치**: 웹캠 (선택 사항이나 권장)

### API 키 설정
프로젝트 루트 디렉토리의 `.env` 파일에 OpenAI API 키가 설정되어 있어야 합니다.
```env
OPENAI_API_KEY=your_openai_api_key_here
POSTGRES_PASSWORD=your_db_password
DB_HOST=localhost
DB_NAME=interview_db
```

## 3. 프로그램 실행 방법

### 의존성 라이브러리 설치
```bash
pip install -r requirements.txt
```

### 서버 실행
```bash
python server.py
```
서버가 실행되면 자동으로 브라우저가 열리며 `http://localhost:5000` 접속을 시도합니다.

## 4. 모델 및 라이브러리

### AI 모델
- **GPT-4o (OpenAI)**: 면접 질문 생성, 답변 평가, 최종 면접 결과 분석 수행
- **Whisper-1 (OpenAI)**: 지원자의 음성 답변을 텍스트로 변환 (STT)

### 주요 라이브러리
- **FastAPI**: 웹 서버 프레임워크
- **psycopg2**: PostgreSQL 데이터베이스 연동
- **pypdf**: PDF 이력서 텍스트 추출
- **uvicorn**: ASGI 서버 실행

## 5. 주요 기능 사용법

### [지원자]
1. **회원가입 및 로그인**: '면접자' 유형으로 가입합니다.
2. **공고 지원**: 대시보드에서 원하는 공고를 선택하고 '지원하기'를 클릭합니다.
3. **이력서 등록**: PDF 형식의 이력서를 업로드합니다.
4. **면접 진행**: 
    - AI 면접관이 질문을 하면 마이크를 통해 답변합니다.
    - 답변 완료 후 '제출' 버튼을 누르면 다음 질문으로 넘어갑니다.
    - 총 12단계(자기소개, 직무 질문, 인성 질문, 마무리)로 진행됩니다.
5. **결과 확인**: 면접 종료 후 '내 면접 기록' 메뉴에서 분석 결과를 확인할 수 있습니다.

### [관리자]
1. **공고 관리**: 새로운 면접 공고를 등록하거나 수정/삭제할 수 있습니다.
2. **지원자 현황**: 면접을 완료한 지원자들의 리스트를 확인하고, 상세 평가 결과(기술 점수, 문제해결력 등)를 조회할 수 있습니다.

## 6. 파일 구조 설명
- `server.py`: FastAPI 기반의 메인 백엔드 서버 로직
- `index.html`: 메인 웹 인터페이스 구조
- `styles.css`: 웹 디자인 및 레이아웃 스타일
- `app.js`: 프론트엔드 비즈니스 로직 및 API 연동
- `create_table.py`: DB 초기 테이블 생성 스크립트
- `uploads/`: 업로드된 이력서 및 녹음된 답변 파일 저장 경로
- `check_columns.py`: DB 스키마 확인을 위한 유틸리티
