# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK)

본 가이드북은 웹 기반 AI 면접 시뮬레이션 프로그램의 개요, 환경 설정, 실행 방법, 및 주요 기능을 설명합니다.

## 1. 개요 (Overview)
이 프로그램은 사용자가 AI 면접관과 모의 면접을 진행할 수 있는 웹 애플리케이션입니다. 지원자는 채용 공고를 확인하고, 이력서를 업로드한 후, 화상 면접(카메라/마이크)을 진행할 수 있습니다. 관리자는 채용 공고를 관리하고 지원자 현황을 파악할 수 있습니다.

### 주요 특징
- **회원가입 및 로그인**: 지원자 및 관리자 계정 생성
- **공고 관리 (관리자)**: 채용 공고(직무 포함) 등록, 수정, 삭제
- **지원 및 면접 (지원자)**: 이력서(PDF) 업로드, 화상 장치 테스트, AI 면접 진행
- **결과 확인**: 면접 결과 시각화 (차트)

## 2. 환경 설정 (Environment Setup)
이 프로그램은 **Python (Flask)** 백엔드와 **Vanilla JS** 프론트엔드로 구성되어 있으며, **PostgreSQL** 데이터베이스를 사용합니다.

### 필수 요구 사항
- Python 3.8 이상
- PostgreSQL 데이터베이스 서버
- 웹 브라우저 (Chrome, Edge 등 Camera/Mic 권한 필요)

### 필수 라이브러리
`server.py` 실행을 위해 다음 파이썬 패키지가 필요합니다.
```bash
pip install flask psycopg2-binary python-dotenv werkzeug
```
*(참고: `werkzeug`는 Flask 설치 시 자동으로 설치됩니다.)*

### 환경 변수 (.env)
프로젝트 상위 폴더 또는 실행 폴더에 `.env` 파일이 있어야 하며, DB 접속 정보가 설정되어야 합니다.
```env
POSTGRES_PASSWORD=your_password
```

## 3. 프로그램 실행 방법 (Execution)
터미널에서 `server.py`가 있는 디렉토리로 이동한 후 다음 명령어를 실행합니다.

```bash
python server.py
```

- 서버가 정상적으로 실행되면 브라우저가 자동으로 열리거나 `http://localhost:5000`으로 접속하여 사용할 수 있습니다.
- 초기 실행 시 DB 테이블(`users`, `interview_announcement`, `interview_information`)이 자동으로 생성됩니다.

## 5. 주요 기능 사용법 (User Manual)

### [관리자]
1. **회원가입/로그인**: '관리자' 유형으로 가입 후 로그인합니다.
2. **공고 관리**:
   - '공고 관리' 메뉴에서 '공고 등록' 버튼을 누릅니다.
   - 제목, **직무(Job)**, 내용, 마감일을 입력하고 등록합니다.
   - 등록된 공고는 수정 및 삭제가 가능합니다.

### [지원자]
1. **회원가입/로그인**: '면접자' 유형으로 가입 후 로그인합니다.
2. **공고 조회 및 지원**:
   - '지원 가능한 공고' 목록에서 '확인하기'를 눌러 상세 내용을 봅니다.
   - '지원하기' 버튼을 누릅니다.
3. **면접 준비**:
   - **이력서 업로드**: PDF 형식의 이력서를 선택합니다.
   - **환경 테스트**: 카메라와 마이크 상태를 확인합니다.
4. **면접 진행**:
   - '면접 시작' 버튼을 누르면 이력서가 업로드되고 면접이 시작됩니다.
   - AI의 질문(음성/텍스트)에 대해 답변을 입력합니다. (음성 인식 시뮬레이션)
5. **결과 확인**: 면접 종료 후 분석된 점수를 확인합니다.
