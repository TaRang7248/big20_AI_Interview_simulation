# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK)

## 1. 개요
본 프로젝트는 **웹 기반 AI 면접 시뮬레이션** 프로그램입니다.
지원자는 실제 면접처럼 AI 면접관과 음성으로 대화하며 면접을 진행하고, 관리자는 면접 결과를 분석하여 지원자를 평가할 수 있습니다.
OpenAI GPT-4o와 Whisper 모델을 활용하여 실시간 질문 생성, 음성 인식(STT), 답변 평가를 수행합니다.

## 2. 환경 및 요구사항
- **OS**: Windows (권장), macOS, Linux
- **Python**: 3.8 이상
- **Database**: PostgreSQL (기본 설정: localhost:5432, DB명: `interview_db`)
- **Browser**: Chrome, Edge 등 모던 웹 브라우저 (카메라/마이크 권한 필요)

### 필수 라이브러리
`requirements.txt`에 명시된 주요 라이브러리는 다음과 같습니다.
- `fastapi`: 웹 서버 프레임워크
- `uvicorn`: ASGI 서버
- `psycopg2-binary`: PostgreSQL 연동
- `openai`: AI 모델 연동
- `python-dotenv`: 환경 변수 관리
- `pypdf`: PDF 이력서 텍스트 추출

## 3. 프로그램 실행 방법

### 3.1. 데이터베이스 준비
PostgreSQL이 실행 중이어야 합니다. `.env` 파일에 DB 접속 정보가 올바르게 설정되어 있는지 확인하세요.
최초 실행 시 테이블이 없다면 `server.py` 실행 시 또는 `create_table.py`를 통해 자동으로 생성됩니다.

### 3.2. 서버 실행
터미널에서 다음 명령어를 실행합니다.
```bash
python server.py
```
서버가 시작되면 자동으로 브라우저가 열리며 `http://localhost:5000`으로 접속됩니다.

## 4. 주요 기능 사용법

### 4.1. 지원자 (Applicant)
1. **회원가입/로그인**: '면접자' 유형으로 가입 및 로그인합니다.
2. **이력서 업로드**: 지원하려는 공고를 선택하고 이력서(PDF)를 업로드합니다.
3. **환경 테스트**: 카메라와 마이크가 정상 작동하는지 확인합니다.
4. **면접 진행**:
   - AI 면접관이 자기소개부터 시작하여 직무, 인성 질문을 순차적으로 건넸습니다.
   - 답변은 음성으로 녹음되며, '답변 제출' 버튼을 누르면 STT 변환 및 분석이 이루어집니다.
5. **면접 종료**: 모든 질문이 끝나면 면접이 종료되고 결과 분석이 시작됩니다.

### 4.2. 관리자 (Admin)
1. **회원가입/로그인**: '관리자' 유형으로 가입 및 로그인합니다.
2. **공고 관리**: 새로운 채용 공고를 등록, 수정, 삭제할 수 있습니다.
3. **지원자 현황 확인**:
   - 지원자들의 면접 진행 상황과 결과를 목록으로 확인합니다.
   - '상세보기'를 클릭하면 **합격/불합격 여부**와 **상세 평가(기술, 문제해결, 의사소통, 태도)**를 확인할 수 있습니다.
   - 새롭게 개선된 UI를 통해 각 항목별 점수와 구체적인 평가 내용을 직관적으로 파악할 수 있습니다.

## 5. 파일 구조 설명

- `server.py`: FastAPI 백엔드 메인 서버 파일. API 엔드포인트와 DB 연동, AI 로직이 포함되어 있습니다.
- `index.html`: 메인 웹 페이지 (SPA 구조).
- `app.js`: 프론트엔드 로직. 라우팅, API 호출, 오디오 녹음 처리 등을 담당합니다.
- `styles.css`: 전체 스타일시트.
- `create_table.py`: 데이터베이스 테이블 초기화 스크립트.
- `setup_interview_data.py`: 초기 면접 질문 데이터 셋업 스크립트.
- `uploads/`: 업로드된 이력서 및 오디오 파일 저장소.

## 6. 사용 모델 및 기술
- **LLM**: OpenAI GPT-4o (질문 생성, 답변 평가, 결과 분석)
- **STT**: OpenAI Whisper-1 (음성 인식)
- **TTS**: Web Speech API (브라우저 내장 음성 합성)
