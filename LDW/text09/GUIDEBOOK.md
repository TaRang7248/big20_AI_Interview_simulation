# AI 면접 시뮬레이션 가이드북 (Guidebook)

## 1. 개요 (Overview)
본 프로젝트는 웹 기반의 AI 면접 시뮬레이션 프로그램입니다. 사용자는 면접관(AI)과 상호작용하며 실전과 같은 면접 경험을 할 수 있습니다. 
주요 기능으로는 회원가입/로그인, 공고 관리(관리자), 면접 진행(지원자), 결과 분석 등이 있습니다.

## 2. 환경 설정 (Environment Setup)

### 필수 요구 사항
- **Python 3.8+**
- **모던 브라우저** (Chrome, Edge 등)
- **PostgreSQL** (데이터베이스 연결 필요)

### 설치 및 실행 준비
1. 프로젝트 폴더로 이동합니다:
   ```bash
   cd C:\big20\big20_AI_Interview_simulation\LDW\text09
   ```
2. 필요한 Python 라이브러리를 설치합니다 (필요한 경우):
   ```bash
   pip install flask psycopg2-binary python-dotenv
   ```
3. `.env` 파일을 프로젝트 루트에 생성하고 데이터베이스 비밀번호를 설정합니다 (기본값: `013579`).

## 3. 프로그램 실행 방법 (How to Run)

서버를 실행하려면 아래 명령어를 터미널에 입력하세요:

```bash
python server.py
```

- 실행 후 브라우저가 자동으로 열리며 `http://localhost:5000`으로 접속됩니다.
- 만약 브라우저가 열리지 않으면 수동으로 주소를 입력하여 접속하세요.

## 4. 사용 라이브러리 (Libraries Used)

### Backend
- **Flask**: 웹 서버 프레임워크
- **psycopg2**: PostgreSQL 데이터베이스 어댑터
- **python-dotenv**: 환경 변수 관리

### Frontend
- **HTML5 / CSS3**: 구조 및 스타일링
- **Vanilla JavaScript (ES6+)**: 프론트엔드 로직
- **Google Fonts**: 폰트 (Noto Sans KR)

## 5. 주요 기능 사용법 (Key Features)

### [공통] 회원가입 및 로그인
- 초기 화면에서 회원가입을 통해 계정을 생성합니다.
- '면접자' 또는 '관리자' 유형을 선택할 수 있습니다.
- 로그인 후 각 역할에 맞는 대시보드로 이동합니다.

### [관리자] 공고 관리
- **공고 등록**: '공고 관리' 메뉴에서 '공고 등록' 버튼을 눌러 제목, **공고 내용**, 마감일을 입력하고 공고를 생성합니다.
- **공고 수정/삭제**: 등록된 공고 리스트에서 '수정' 또는 '삭제' 버튼을 사용하여 관리할 수 있습니다.
- **지원자 현황**: 지원자들의 면접 결과 및 점수를 확인할 수 있습니다 (구현 예정/Mock 데이터).

### [지원자] 면접 진행
- **공고 지원**: 대시보드에서 지원 가능한 공고를 확인하고 '지원하기'를 클릭합니다.
- **환경 점검**: 카메라와 마이크 작동 여부를 테스트합니다.
- **면접 시작**: AI 면접관의 질문에 음성으로 답변합니다 (현재 텍스트 입력 시뮬레이션).
- **결과 확인**: 면접 종료 후 기술, 문제해결, 의사소통, 태도 역량별 점수와 합격 여부를 확인합니다.

### [공통] 마이페이지
- **내 정보 수정**: 비밀번호 확인 후 이메일, 주소, 전화번호를 수정할 수 있습니다.
- **비밀번호 변경**: 현재 비밀번호 확인 과정을 거친 후 새로운 비밀번호로 변경할 수 있습니다.
