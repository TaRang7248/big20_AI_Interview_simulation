# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK)

본 가이드북은 웹 기반 AI 면접 시뮬레이션 프로그램의 개요, 설치 및 실행 방법, 주요 기능, 파일 구조 등을 설명합니다.

## 1. 개요 (Overview)
이 프로그램은 사용자가 가상의 AI 면접관과 인터뷰를 진행하며 면접 역량을 키울 수 있도록 돕는 웹 애플리케이션입니다.
Python Flask 서버와 순수 JavaScript(Vanilla JS) 프론트엔드로 구성되어 있으며, SQLite를 사용하여 회원 정보를 관리합니다.

### 주요 특징
- **회원 관리**: 회원가입, 로그인, 정보 수정, **비밀번호 변경** 기능 제공
- **면접 시뮬레이션**: 단계별(자기소개, 역량 검증, 기술 면접) 질문 제시 및 타이머 기능
- **음성 인식/합성**: (브라우저 지원 시) 질문 읽어주기(TTS) 및 답변 음성 인식(STT) 시뮬레이션
- **결과 분석**: 면접 완료 후 점수 및 합격 예측 결과 제공
- **관리자 모드**: 공고 관리 및 지원자 현황 조회 기능

## 2. 환경 설정 (Environment)
본 프로그램은 다음 환경에서 실행을 권장합니다.

- **OS**: Windows, macOS, Linux (플랫폼 독립적)
- **Language**: Python 3.8 이상
- **Browser**: Google Chrome, MS Edge 등 최신 브라우저 (권한 허용 필요)

## 3. 프로그램 실행 방법 (Execution)

### 1단계: 필수 라이브러리 설치
터미널(CMD 또는 PowerShell)에서 다음 명령어를 실행하여 필요한 패키지를 설치합니다.

```bash
pip install flask
```
*(참고: SQLite3는 Python 기본 내장 라이브러리이므로 별도 설치가 필요 없습니다.)*

### 2단계: 서버 실행
`server.py`가 있는 디렉토리로 이동한 후 다음 명령어를 실행합니다.

```bash
python server.py
```

### 3단계: 접속
서버가 정상적으로 실행되면 브라우저가 자동으로 열리거나, 수동으로 주소창에 `http://localhost:5000`을 입력하여 접속합니다.

## 4. 사용 라이브러리 (Libraries)

- **Backend**
  - `Flask`: 웹 서버 프레임워크 (API 제공 및 정적 파일 서빙)
  - `sqlite3`: 경량 관계형 데이터베이스 (내장)
  - `os`, `webbrowser`, `threading`: 시스템 제어 및 유틸리티

- **Frontend**
  - `HTML5`, `CSS3`: 구조 및 스타일링 (Responsive Design)
  - `JavaScript (ES6+)`: 클라이언트 로직 (SPA 라우팅, 비동기 통신)
  - `Google Fonts (Noto Sans KR)`: 폰트 리소스

## 5. 주요 기능 사용법

### 5.1 회원가입 및 로그인
1. 메인 화면에서 '회원가입' 링크를 클릭하여 계정을 생성합니다.
2. 생성한 아이디와 비밀번호로 로그인합니다.

### 5.2 면접 체험 (지원자)
1. 로그인 후 대시보드에서 '지원 가능한 공고' 목록을 확인합니다.
2. 공고의 '지원하기' 버튼을 클릭하여 면접 준비 페이지로 이동합니다.
3. 카메라/마이크 테스트를 진행합니다 (브라우저 권한 허용 필요).
4. '면접 시작' 버튼을 눌러 인터뷰를 진행합니다.
5. AI의 질문을 듣고 답변을 입력하거나 말합니다(STT 시뮬레이션).
6. 모든 단계가 끝나면 결과를 확인합니다.

### 5.3 정보 수정 및 비밀번호 변경
1. 로그인 후 네비게이션 바의 사용자 이름(또는 '내 정보')을 클릭합니다.
2. 현재 비밀번호를 입력하여 본인 확인을 진행합니다.
3. 정보 수정 페이지에서 이메일, 주소 등을 수정할 수 있습니다.
4. **'비밀번호 변경'** 버튼을 클릭하여 새로운 비밀번호를 설정할 수 있습니다.

### 5.4 관리자 모드
1. 회원가입 시 '관리자' 유형을 선택하거나, DB에서 유형을 변경한 계정으로 로그인합니다.
2. 공고를 추가하거나 지원자들의 면접 결과를 조회할 수 있습니다.

```
    └── membership_information.db  # SQLite 데이터베이스 파일 (자동 생성)
```
