# 📘 AI 면접 시뮬레이션 가이드북 (AI Interview Simulation Guidebook)

## 1. 개요 (Overview)
본 프로젝트는 **웹 기반 AI 면접 시뮬레이션 플랫폼**입니다. 사용자는 회원가입 후 직무별 공고를 확인하고 지원할 수 있으며, AI 면접관과 음성 및 텍스트 기반으로 가상 면접을 진행할 수 있습니다. 면접 종료 후에는 기술, 문제 해결, 의사소통, 태도 등 4가지 영역에 대한 평가 결과를 확인할 수 있습니다.

### 주요 특징
- **회원 관리**: 지원자 회원가입/로그인 및 내 정보 수정 기능.
- **채용 공고**: 관리자(Admin)에 의한 공고 등록/수정/삭제 및 지원자의 공고 조회.
- **이력서 관리**: PDF 이력서 업로드 및 파싱 기능을 통한 맞춤형 질문 생성 준비.
- **AI 면접**: 카메라/마이크 연동, TTS(음성 합성) 질문 제시, 타이머 제한 답변.
- **결과 분석**: 면접 답변에 대한 자동 평가 및 점수화 (Simulation).

---

## 2. 개발 환경 (Environment)
이 프로그램은 **Python** 기반의 백엔드와 **Vanilla JavaScript** 기반의 프론트엔드로 구성되어 있습니다.

- **OS**: Windows (테스트 환경)
- **Language**: Python 3.9+, HTML5, CSS3, JavaScript (ES6+)
- **Database**: PostgreSQL
- **Browser**: Chrome, Edge 등 최신 브라우저 (카메라/마이크 권한 필요)

---

## 3. 프로그램 실행 방법 (How to Run)

### 3.1 사전 준비
1. **Python 설치**: Python이 설치되어 있어야 합니다.
2. **PostgreSQL 설치**: 데이터베이스 서버가 실행 중이어야 합니다.
   - DB명: `interview_db`
   - 사용자: `postgres`
   - 기본 비밀번호: `013579` (또는 `.env` 설정)
3. **라이브러리 설치**:
   프로젝트 폴더 내에서 터미널을 열고 다음 명령어를 실행하여 필수 라이브러리를 설치합니다.
   ```bash
   pip install -r requirements.txt
   ```

### 3.2 서버 실행
터미널에서 `server.py`를 실행합니다.
```bash
python server.py
```
- 서버가 정상적으로 시작되면 `Serving on http://localhost:5000` 메시지가 출력됩니다.
- 브라우저가 자동으로 열리며 `http://localhost:5000`으로 접속됩니다.

## 5. 주요 기능 사용법 (User Manual)

### 5.1 회원가입 및 로그인
- 초기 실행 시 로그인 화면이 나타납니다.
- '회원가입' 버튼을 눌러 계정을 생성합니다.
    - **면접자**: 일반 사용자로서 채용 공고에 지원할 수 있습니다.
    - **관리자**: 공고를 등록하고 관리할 수 있습니다.
- 회원가입 후 로그인하면 대시보드로 이동합니다.

### 5.2 (면접자) 채용 공고 및 지원
1. **대시보드**: 등록된 채용 공고 목록을 확인합니다.
2. **상세 보기**: '확인하기' 버튼을 눌러 공고 상세 내용을 봅니다.
3. **지원하기**: '지원하기'를 누르면 면접 준비 단계로 넘어갑니다.
4. **이력서 업로드**: PDF 형식의 이력서를 업로드합니다.
5. **환경 테스트**: 카메라와 마이크가 정상 작동하는지 테스트합니다.
6. **면접 시작**: 준비가 완료되면 활성화된 '면접 시작' 버튼을 누릅니다.

### 5.3 AI 면접 진행
- **대기/질문**: AI 면접관이 음성(TTS)과 텍스트로 질문을 제시합니다.
- **답변**: 화면 중앙의 텍스트 박스에 답변을 입력하거나, 음성으로 답변합니다(시뮬레이션). 60초의 제한 시간이 있습니다.
- **제출**: '답변 완료' 버튼을 누르거나 시간이 종료되면 다음 질문으로 넘어갑니다.
- 총 5개의 질문이 진행됩니다.

### 5.4 결과 확인
- 면접이 종료되면 자동으로 결과 페이지로 이동합니다.
- 기술, 문제 해결, 의사소통, 태도 4가지 영역의 점수 그래프와 합격/불합격 예측 여부를 확인할 수 있습니다.

### 5.5 (관리자) 공고 관리
- 관리자 계정으로 로그인 시 '공고 관리' 메뉴가 보입니다.
- **공고 등록**: 새로운 채용 공고를 작성하여 게시할 수 있습니다.
- **공고 수정/삭제**: 기존 공고를 수정하거나 삭제할 수 있습니다.
