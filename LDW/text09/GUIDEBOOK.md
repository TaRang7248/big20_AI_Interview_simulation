
# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK.md)

## 1. 개요 (Overview)
본 프로그램은 웹 기반의 AI 면접 시뮬레이션 시스템입니다. 사용자는 회원가입 후 로그인을 통해 모의 면접을 진행할 수 있으며, AI 면접관의 질문에 답변하고 결과를 확인할 수 있습니다.

## 2. 환경 설정 (Environment Setup)

### 필수 요구 사항
*   **OS:** Windows 10/11
*   **Python:** 3.8 이상
*   **Browser:** Chrome, Edge (최신 버전 권장)

### 설치 방법
1.  **소스 코드 다운로드:** 제공된 압축 파일을 해제합니다.
2.  **가상 환경 설정 (선택 사항):**
    ```bash
    python -m venv venv
    venv\Scripts\activate
    ```
3.  **라이브러리 설치:**
    ```bash
    pip install -r requirements.txt
    ```
    *(만약 requirements.txt가 없다면 아래 '3. 사용 라이브러리' 항목을 참고하여 직접 설치하세요.)*
4.  **환경 변수 설정 (.env):**
    프로젝트 루트 디렉토리에 `.env` 파일이 존재해야 하며, 데이터베이스 접속 정보 등이 포함되어야 합니다.

## 3. 사용 라이브러리 (Libraries Used)

본 프로젝트는 다음과 같은 주요 Python 라이브러리를 사용합니다.

*   **Flask:** 웹 서버 프레임워크
*   **psycopg2:** PostgreSQL 데이터베이스 연동 어댑터
*   **python-dotenv:** 환경 변수 관리

설치 명령어:
```bash
pip install flask psycopg2-binary python-dotenv
```

## 4. 프로그램 실행 방법 (How to Run)

1.  명령 프롬프트(CMD) 또는 터미널을 엽니다.
2.  프로젝트 디렉토리로 이동합니다: `cd C:\big20\big20_AI_Interview_simulation\LDW\text09`
3.  서버 실행 명령어를 입력합니다:
    ```bash
    python server.py
    ```
4.  자동으로 브라우저가 열리며 `http://localhost:5000`에 접속됩니다.

## 5. 주요 기능 사용법 (Key Features)

### 5.1 회원가입
1.  메인 화면에서 **[회원가입]** 링크를 클릭합니다.
2.  ID, 비밀번호, 이름, 생년월일, 성별, 이메일, 주소, 전화번호를 입력합니다.
3.  **[가입하기]** 버튼을 클릭하여 완료합니다.

### 5.2 로그인
1.  가입한 ID와 비밀번호를 입력하고 **[로그인]** 버튼을 클릭합니다.

### 5.3 마이 페이지 (내 정보 수정)
1.  로그인 후 상단 네비게이션 바에서 **[내 정보]**를 클릭합니다.
2.  비밀번호를 재입력하여 본인 확인을 거칩니다.
3.  이메일, 주소, 전화번호 등을 수정하거나 **[비밀번호 변경]** 버튼을 눌러 비밀번호를 변경할 수 있습니다.

### 5.4 면접 진행
1.  지원자 대시보드에서 **[지원하기]** 버튼을 클릭합니다.
2.  환경 점검(카메라/마이크)을 진행한 후 **[면접 시작]**을 누릅니다.
3.  AI 면접관의 질문에 음성 또는 텍스트로 답변합니다.
4.  모든 단계가 끝나면 결과 페이지로 이동합니다.
