# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK)

## 1. 개요 (Overview)
본 프로그램은 웹 기반의 AI 면접 시뮬레이션 플랫폼입니다. 사용자는 회원가입 및 로그인을 통해 가상 면접에 참여할 수 있으며, AI 면접관과 음성 대화를 통해 실제 면접과 유사한 경험을 할 수 있습니다. 관리자는 채용 공고를 관리하고 지원자의 결과를 확인할 수 있습니다.

## 2. 환경 설정 (Environment Setup)
본 프로그램은 Python 기반의 웹 서버(Flask)와 바닐라 자바스크립트(Vanilla JS) 프론트엔드로 구성되어 있습니다.

### 필수 요구 사항
- **OS**: Windows (본 가이드 기준)
- **Python**: 3.8 이상
- **Browser**: Chrome, Edge 등 최신 브라우저 (카메라/마이크 권한 필요)
- **Database**: PostgreSQL (코드 상 `psycopg2` 사용, 설정에 따라 로컬 DB 사용 가능)

### 3. 프로그램 실행 방법 (How to Run)
1. **필수 라이브러리 설치**:
   터미널에서 아래 명령어를 입력하여 필요한 패키지를 설치합니다.
   ```bash
   pip install flask psycopg2 binary python-dotenv
   ```
   *(참고: `psycopg2` 설치가 어려울 경우 `psycopg2-binary`를 권장합니다.)*

2. **서버 실행**:
   프로젝트 폴더(`C:\big20\big20_AI_Interview_simulation\LDW\text09`)로 이동 후 아래 명령어를 실행합니다.
   ```bash
   python server.py
   ```

3. **접속**:
   서버가 정상적으로 실행되면 브라우저가 자동으로 열리거나, 수동으로 주소창에 `http://localhost:5000`을 입력하여 접속합니다.

## 4. 사용 라이브러리 (Libraries)
### Backend (Python)
- **Flask**: 웹 서버 프레임워크
- **psycopg2**: PostgreSQL 데이터베이스 연동 어댑터
- **python-dotenv**: 환경 변수(.env) 관리
- **webbrowser**: 서버 실행 시 브라우저 자동 오픈

### Frontend
- **Vanilla JS (ES6+)**: 별도의 프레임워크 없이 순수 자바스크립트로 구현
- **Google Fonts**: Noto Sans KR 폰트 사용
- **Web Speech API**: 브라우저 내장 음성 인식(STT) 및 음성 합성(TTS)

## 5. 주요 기능 사용법 (Key Features)
### A. 회원가입 및 로그인
- **회원가입**: 아이디, 비밀번호, 이름, 생년월일, 성별, 이메일, 주소, 전화번호를 입력하여 가입합니다. (전화번호는 3개 필드로 나누어 입력)
- **로그인**: 가입한 아이디와 비밀번호로 접속합니다.
- **정보 수정**: '내 정보' 메뉴에서 비밀번호 확인 후 이메일, 주소, 전화번호를 수정하거나 비밀번호를 변경할 수 있습니다.

### B. 면접 진행 (지원자)
1. **공고 지원**: 대시보드에서 원하는 공고의 '지원하기' 버튼을 클릭합니다.
2. **환경 테스트**: 카메라와 마이크 권한을 허용하고 정상 작동을 확인합니다.
3. **면접 시작**: AI 면접관의 질문(음성+텍스트)을 듣고 본인의 답변을 음성으로 말합니다.
4. **결과 확인**: 면접 종료 후 영역별 점수와 합격 여부 예측 결과를 확인합니다.

### C. 관리자 기능
- **공고 관리**: 새로운 채용 공고를 등록하거나 기존 공고를 관리합니다.
- **지원자 현황**: 지원자들의 면접 점수와 합격 여부를 목록으로 조회합니다.
