# AI 면접 프로그램 가이드북 (User Guide)

본 문서는 웹 기반 AI 면접 시뮬레이션 프로그램의 설치, 실행 및 사용 방법에 대해 설명합니다.

## 1. 개요
이 프로그램은 구직자가 실제 면접 상황을 대비하여 AI와 모의 면접을 진행할 수 있는 웹 애플리케이션입니다. 카메라와 마이크를 사용하여 실제 면접과 유사한 환경을 제공하며, 면접 후 결과를 확인할 수 있습니다.

## 2. 설치 및 실행 방법
본 프로그램은 별도의 서버 설치 없이 웹 브라우저만으로 실행 가능합니다.

1. **파일 준비**: 제공된 소스 코드 폴더(`text09`)를 준비합니다.
2. **실행**: 폴더 내의 `index.html` 파일을 더블 클릭하거나, 웹 브라우저(Chrome, Edge 등)로 드래그하여 엽니다.
   - *권장 사항*: 원활한 카메라/마이크 사용을 위해 VS Code의 'Live Server' 확장 프로그램을 사용하는 것을 권장합니다. (보안 정책상 로컬 파일 직접 실행 시 카메라 권한이 차단될 수 있음)

## 3. 주요 기능 및 사용법

### 3.1 회원가입 (Sign Up)
처음 사용자는 회원가입을 통해 계정을 생성해야 합니다.
1. 로그인 화면 하단의 **'회원가입'** 링크를 클릭합니다.
2. 다음 정보를 입력합니다:
   - **아이디**: 로그인에 사용할 ID
   - **비밀번호**: 계정 비밀번호
   - **이름**: 사용자 실명
   - **생년월일**: [추가됨] 사용자의 생년월일
   - **성별**: [추가됨] 남성/여성 선택
   - **이메일**: [추가됨] 연락 가능한 이메일 주소
   - **주소**: [추가됨] 거주지 주소
   - **전화번호**: [추가됨] 연락처
   - **회원 유형**: '면접자' 선택 (관리자 기능 테스트 시 '관리자' 선택)
3. **'가입하기'** 버튼을 클릭하면 가입이 완료되며 로그인 화면으로 이동합니다.

### 3.2 로그인 (Login)
1. 가입한 **아이디**와 **비밀번호**를 입력합니다.
2. **'로그인'** 버튼을 클릭합니다.
   - *테스트 계정*: ID: `test`, PW: `1234` (면접자) / ID: `admin`, PW: `admin` (관리자)

### 3.3 면접 진행 (Applicant Mode)
1. **대시보드**: 로그인 후 지원 가능한 공고 목록이 표시됩니다.
2. **지원하기**: 원하는 공고의 **'지원하기'** 버튼을 클릭합니다.
3. **환경 설정**:
   - 이력서 파일을 업로드합니다.
   - **'장치 테스트 실행'** 버튼을 눌러 카메라와 마이크 권한을 허용하고 작동을 확인합니다.
4. **면접 시작**: 모든 준비가 완료되면 **'면접 시작'** 버튼이 활성화됩니다.
5. **면접 진행**:
   - AI 면접관이 질문을 읽어줍니다 (TTS).
   - 질문이 끝난 후 타이머가 시작되며, 마이크를 통해 답변하거나 텍스트로 입력할 수 있습니다.
   - **Phase 1 (자기소개) -> Phase 2 (역량 검증) -> Phase 3 (기술 면접)** 순으로 진행됩니다.
6. **결과 확인**: 면접 종료 후 영역별 점수와 합격 여부 예측 결과를 확인할 수 있습니다.

### 3.4 관리자 모드 (Admin Mode)
관리자 계정(`admin`)으로 로그인하면 공고를 등록하거나 지원자들의 면접 결과를 조회할 수 있습니다.

## 4. 문제 해결
- **카메라/마이크가 작동하지 않아요**: 브라우저 주소창 설정에서 사이트 설정으로 이동하여 카메라/마이크 권한이 '허용'되어 있는지 확인하세요. 로컬 파일(`file://`) 실행 시 제한될 수 있으므로 로컬 웹 서버(`http://localhost`) 환경에서 실행하는 것을 권장합니다.
- **음성이 들리지 않아요**: PC의 스피커 볼륨을 확인해주시고, 브라우저 탭이 음소거 상태인지 확인해주세요.
