# 📘 AI 면접 시뮬레이션 가이드북 (AI Interview Simulation Guidebook)

## 1. 개요 (Overview)
본 프로젝트는 **웹 기반 AI 면접 시뮬레이션 플랫폼**입니다. 사용자는 실제 면접과 유사한 환경에서 AI 면접관과 질의응답을 진행하고, 자신의 면접 실력을 객관적으로 평가받을 수 있습니다.

### 주요 기능
- **회원 관리**: 지원자/관리자 구분을 통한 회원가입 및 로그인, 개인정보 수정.
- **채용 공고**: 관리자의 공고 등록/관리 및 지원자의 실시간 공고 확인 및 지원.
- **이력서 기반 맞춤형 질문**: PDF 이력서를 업로드하면 텍스트를 분석하여 사용자 맞춤형 질문 생성.
- **AI 인터랙션**: TTS(음성 출력)와 STT(음성 인식)를 활용한 실감 나는 면접 인터페이스.
- **결과 리포트**: 면접 종료 후 기술 역량, 문제 해결력, 의사소통, 태도 등 다각도 분석 결과 제공.

---

## 2. 개발 환경 및 요구 사항 (Environment)
프로그램의 정상적인 작동을 위해 아래 환경을 권장합니다.

- **운영체제(OS)**: Windows 10/11
- **언어 및 프레임워크**:
  - Python 3.9 이상
  - Flask (백엔드 서버)
  - Vanilla JavaScript (프론트엔드)
- **데이터베이스**: PostgreSQL
- **권한 설정**: 브라우저에서 카메라 및 마이크 접근 권한 허용이 필요합니다.

---

## 3. 설치 및 실행 방법 (Installation & Run)

### 3.1 라이브러리 설치
프로젝트 루트 폴더에서 터미널을 열고 필요한 종속성을 설치합니다.
```bash
pip install -r requirements.txt
```

### 3.2 데이터베이스 설정
- **기본 접속 정보**:
  - Host: `127.0.0.1`
  - Port: `5432`
  - Database: `interview_db`
  - User: `postgres`
  - Password: `013579` (또는 환경 변수 `POSTGRES_PASSWORD` 설정)

### 3.3 서버 실행
터미널에서 서버를 실행합니다.
```bash
python server.py
```
서버 실행 시 자동으로 브라우저가 열리며 [http://localhost:5000](http://localhost:5000)으로 접속됩니다.

## 4. 사용 라이브러리 (Libraries)
| 라이브러리 | 용도 |
| :--- | :--- |
| **Flask** | 웹 애플리케이션 프레임워크 및 API 서버 구축 |
| **Psycopg2** | PostgreSQL 데이터베이스 연동 및 쿼리 실행 |
| **Werkzeug** | 비밀번호 해싱(`pbkdf2:sha256`) 및 보안 유틸리티 |
| **PyPDF** | PDF 이력서 파일 읽기 및 텍스트 추출 |
| **Python-Dotenv** | 설정값 및 비밀번호 관리를 위한 환경 변수 로드 |

## 5. 주요 기능 사용법 (User Manual)

### 5.1 로그인 및 회원가입
- **로그인**: 가입된 아이디와 비밀번호로 접속합니다. 초기에는 `check_users.py`를 통해 계정 상태를 확인할 수 있습니다.
- **회원 유형**: '면접자'는 면접 응시가 가능하며, '관리자'는 공고를 등록하고 지원자를 관리할 수 있습니다.

### 5.2 면접 준비 (지원하기)
1. **공고 선택**: 대시보드에서 원하는 공고의 '확인하기' 버튼을 누릅니다.
2. **이력서 업로드**: 반드시 **PDF 형식**의 이력서를 등록해야 합니다.
3. **장치 테스트**: '장치 테스트' 버튼을 눌러 카메라와 마이크 소통이 가능한지 확인합니다.

### 5.3 AI 면접 진행
- AI 면접관이 질문을 읽어주며 화면에 텍스트가 표시됩니다.
- 각 질문당 **60초**의 시간 제한이 있으며, 음성이나 텍스트로 답변을 입력할 수 있습니다.
- 총 5단계의 질의응답이 진행된 후 자동으로 분석 단계로 넘어갑니다.
