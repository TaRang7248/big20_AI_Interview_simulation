# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK.md)

## 1. 개요 (Overview)
본 프로그램은 웹 기반의 AI 면접 시뮬레이션 시스템입니다. 사용자는 회원가입 후 로그인을 통해 모의 면접을 진행할 수 있으며, AI 면접관의 질문에 답변하고 결과를 확인할 수 있습니다. 또한 관리자 기능을 통해 면접 공고를 관리하고 지원자 현황을 파악할 수 있습니다.

## 2. 환경 설정 (Environment Setup)

### 필수 요구 사항
*   **OS:** Windows 10/11
*   **Python:** 3.8 이상
*   **Browser:** Chrome, Edge (최신 버전 권장)

### 설치 방법
1.  **소스 코드 다운로드:** 제공된 압축 파일을 해제합니다.
2.  **가상 환경 설정 (선택 사항):**
    ```bash
    python -m venv venv
    venv\Scripts\activate
    ```
3.  **라이브러리 설치:**
    ```bash
    pip install -r requirements.txt
    ```
    *(만약 requirements.txt가 없다면 아래 '3. 사용 라이브러리' 항목을 참고하여 직접 설치하세요.)*
4.  **환경 변수 설정 (.env):**
    프로젝트 루트 디렉토리에 `.env` 파일이 존재해야 하며, 데이터베이스 접속 정보 등이 포함되어야 합니다.

## 3. 사용 라이브러리 (Libraries Used)

본 프로젝트는 다음과 같은 주요 Python 라이브러리를 사용합니다.

*   **Flask:** 웹 서버 프레임워크 (백엔드 API 및 정적 파일 서빙)
*   **psycopg2:** PostgreSQL 데이터베이스 연동 어댑터
*   **python-dotenv:** 환경 변수 관리 (.env 파일 로드)

**설치 명령어:**
```bash
pip install flask psycopg2-binary python-dotenv
```

## 4. 프로그램 실행 방법 (How to Run)

1.  명령 프롬프트(CMD) 또는 터미널을 엽니다.
2.  프로젝트 디렉토리로 이동합니다: 
    `cd C:\big20\big20_AI_Interview_simulation\LDW\text09`
3.  서버 실행 명령어를 입력합니다:
    ```bash
    python server.py
    ```
4.  서버가 실행되면 자동으로 브라우저가 열리며 `http://localhost:5000`에 접속됩니다. (자동으로 열리지 않을 경우 브라우저 주소창에 직접 입력)

## 5. 주요 기능 사용법 (Key Features)

### 5.1 회원가입 및 로그인
*   **회원가입:** 메인 화면의 [회원가입] 링크를 통해 ID, 비밀번호, 이름, 생년월일, 성별, 이메일, 주소, 전화번호를 입력하여 가입합니다.
*   **로그인:** 가입한 ID와 비밀번호로 로그인합니다.

### 5.2 마이 페이지 (내 정보 수정)
1.  로그인 후 상단 네비게이션 바에서 **[내 정보]**를 클릭합니다.
2.  보안을 위해 비밀번호를 재입력하여 본인 확인을 거칩니다.
3.  이메일, 주소, 전화번호를 수정하고 **[정보 수정 완료]**를 클릭하여 저장합니다.
4.  **[비밀번호 변경]** 버튼을 클릭하여 새로운 비밀번호로 변경할 수 있습니다.

### 5.3 면접 진행 (지원자)
1.  대시보드에서 지원 가능한 공고의 **[지원하기]** 버튼을 클릭합니다.
2.  **환경 점검:** 카메라와 마이크 권한을 허용하고 연결 상태를 확인합니다.
3.  **면접 시작:** AI 면접관의 음성 질문을 듣고 답변을 진행합니다. (음성 인식 및 TTS 기능 포함)
4.  **결과 확인:** 면접 종료 후 기술 역량, 문제 해결, 의사 소통, 태도 영역별 점수와 합격 여부 예측 결과를 확인합니다.

### 5.4 관리자 기능
*   관리자 계정으로 로그인 시 공고 관리 및 지원자 현황 조회 메뉴에 접근할 수 있습니다.
