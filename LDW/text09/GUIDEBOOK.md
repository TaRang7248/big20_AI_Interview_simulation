# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK)

이 문서는 AI 기반 면접 시뮬레이션 프로그램의 구성, 설치 및 사용법에 대한 상세 가이드를 제공합니다.

## 1. 프로젝트 개요
본 프로그램은 지원자가 업로드한 이력서와 공고 내용을 바탕으로 AI 면접관이 질문을 생성하고, 지원자의 음성 답변을 분석하여 평가하는 웹 기반 시뮬레이션 도구입니다.

## 2. 개발 환경 및 기술 스택
- **Backend**: Python (FastAPI)
- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **Database**: PostgreSQL
- **AI Models**:
  - **GPT-4o**: 질문 생성 및 답변 평가, 결과 분석
  - **Whisper-1**: 음성 답변의 텍스트 변환 (STT)
- **주요 라이브러리**:
  - `psycopg2-binary`: PostgreSQL 연동
  - `openai`: AI API 인터페이스
  - `python-multipart`: 파일 업로드 처리
  - `pydantic`: 데이터 검증

## 3. 프로그램 실행 방법
1.  **데이터베이스 설정**: PostgreSQL에 `interview_db` 데이터베이스를 생성합니다.
2.  **의존성 설치**: 
    ```bash
    pip install -r requirements.txt
    ```
3.  **테이블 생성**:
    ```bash
    python create_table.py
    ```
4.  **서버 실행**:
    ```bash
    python server.py
    ```
5.  **접속**: 브라우저에서 `http://localhost:5000`으로 접속합니다.

## 4. 주요 기능 사용법
- **회원가입/로그인**: 지원자 또는 관리자로 가입하여 대시보드에 접근합니다.
- **공고 관리 (관리자)**: 면접 공고를 생성하고 지원자 현황 및 평가 결과를 확인합니다.
- **면접 응시 (지원자)**:
  1. 공고 선택 후 이력서(PDF)를 업로드합니다.
  2. 카메라 및 마이크 장치 테스트를 완료합니다.
  3. AI 면접관의 질문에 음성으로 응답합니다. (각 질문당 90초)
  4. 답변 제출 후 AI가 다음 질문을 생성합니다.
- **결과 확인**: 면접 종료 후 기술/문제해결/의사소통/태도 별 점수와 종합 합격 여부를 확인합니다.

## 5. 파일 구조 설명
- `server.py`: FastAPI 서버 로직 및 AI 연동 API.
- `app.js`: 프론트엔드 라우팅 및 면접 상호작용 로직.
- `index.html`: 메인 웹 페이지 구조.
- `styles.css`: 디자인 및 레이아웃 스타일.
- `data.json`: 초기 질문 풀 데이터.
- `uploads/`: 업로드된 이력서 저장 폴더.
- `audio/`: 녹음된 면접 답변 오디오 저장 폴더.

## 6. 주의 사항
- OpenAI API Key가 환경 변수 또는 코드 내에 올바르게 설정되어 있어야 합니다.
- 마이크와 카메라 권한 사용이 허용되어야 정상적인 면접 응시가 가능합니다.
- 면접 답변 시간이 종료되면 현재까지의 음성을 바탕으로 자동 평가가 진행됩니다.
