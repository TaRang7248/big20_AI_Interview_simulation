# AI 면접 시뮬레이션 (AI Interview Simulation) 가이드북

## 1. 개요 (Overview)
본 프로젝트는 **웹 기반 AI 면접 시뮬레이션 플랫폼**입니다. 사용자는 PDF 이력서를 업로드하고, **GPT-4o**와 **Whisper** 기술이 적용된 AI 면접관과 실제 면접처럼 대화를 나눌 수 있습니다. AI는 이력서를 분석하여 맞춤형 질문을 생성하고, 사용자의 음성 답변을 텍스트로 변환(STT)하여 평가한 뒤 꼬리 질문을 이어갑니다.

### 주요 특징
*   **이력서 기반 맞춤 질문**: PDF 이력서를 분석하여 지원자에게 특화된 질문 생성.
*   **음성 대화 인터페이스**: WebRTC를 이용한 화상/음성 녹음 및 Whisper STT, TTS(Text-to-Speech) 지원.
*   **실시간 평가 및 꼬리 질문**: GPT-4o가 답변을 실시간으로 분석하여 평가하고 심층 질문 생성.
*   **관리자 기능**: 면접 공고 관리 및 지원자 현황 조회 가능.

## 2. 개발 환경 (Environment)
본 프로젝트는 Python FastAPI 백엔드와 Vanilla JS 프론트엔드로 구성되어 있습니다.

### 필수 요구 사항
*   **OS**: Windows 10/11
*   **Python**: 3.8 이상
*   **PostgreSQL**: 13 이상 (DB 설정 필요)
*   **Browser**: Chrome, Edge (마이크/카메라 권한 필요)
*   **API Key**: OpenAI API Key (GPT-4o, Whisper 사용)

## 3. 설치 및 실행 방법 (Installation & Usage)

### 3.1. 데이터베이스 설정
PostgreSQL에 `interview_db` 데이터베이스를 생성하고, `.env` 파일에 접속 정보를 설정해야 합니다.

```bash
# PostgreSQL 접속 후
CREATE DATABASE interview_db;
```

### 3.2. 패키지 설치
프로젝트 루트 디렉토리에서 필요한 Python 패키지를 설치합니다.

```bash
pip install -r requirements.txt
```

### 3.3. 환경 변수 설정
`.env` 파일을 생성하거나 수정하여 아래 정보를 입력합니다.
```ini
DB_HOST=localhost
DB_NAME=interview_db
DB_USER=postgres
POSTGRES_PASSWORD=your_password
DB_PORT=5432
OPENAI_API_KEY=your_openai_api_key_here
```

### 3.4. 서버 실행
아래 명령어로 서버를 실행합니다.
```bash
python server.py
```
*   서버가 시작되면 자동으로 웹 브라우저가 열립니다 (`http://localhost:5000`).
*   초기 실행 시 DB 테이블이 자동으로 생성됩니다.

## 5. 주요 기능 사용법 (User Guide)

### 5.1. 회원가입 및 로그인
*   **회원가입**: ID, 비밀번호, 이름, 생년월일 등을 입력하여 가입합니다. '면접자'와 '관리자' 유형을 선택할 수 있습니다.
*   **로그인**: 가입한 계정으로 로그인합니다.

### 5.2. 면접 공고 확인 및 지원 (면접자)
1.  로그인 후 대시보드에서 **지원 가능한 공고** 목록을 확인합니다.
2.  공고를 클릭하여 상세 내용을 확인하고 **[지원하기]** 버튼을 누릅니다.
3.  **이력서 업로드**: 본인의 이력서(PDF)를 선택하여 업로드합니다.
4.  **환경 테스트**: 카메라와 마이크 연결 상태가 자동으로 확인됩니다.
5.  **면접 시작**: 준비가 완료되면 [면접 시작] 버튼을 누릅니다.

### 5.3. 실전 AI 면접
1.  **질문 청취**: AI 면접관이 첫 질문을 음성와 텍스트로 제시합니다.
2.  **답변 하기**: 질문이 끝나면 자동으로 녹음이 시작됩니다. 카메라를 보고 답변을 말하세요.
    *   **실시간 STT**: 말하는 내용이 화면 하단 텍스트 상자에 실시간으로 표시됩니다.
3.  **답변 완료**: 답변을 마치면 **[답변 완료]** 버튼을 누르거나 타이머가 종료될 때까지 기다립니다.
    *   **답변 없음**: 만약 아무 말도 하지 않고 [답변 완료]를 누르면 "답변 없음"으로 처리되어 다음 질문으로 넘어갑니다.
4.  **다음 질문**: AI가 답변을 분석하고 다음 꼬리 질문을 생성합니다. (약 3~5초 소요)
5.  이 과정을 반복하여 면접을 진행합니다.

### 5.4. 공고 관리 (관리자)
*   관리자 계정으로 로그인하면 **공고 등록/수정/삭제**가 가능합니다.
*   지원자들의 면접 기록(현황)을 조회할 수 있습니다.

## 6. 문제 해결 (Troubleshooting)

### Q. 음성 인식이 안 돼요.
*   브라우저의 마이크 권한이 허용되어 있는지 확인하세요.
*   조용한 환경에서 또렷하게 말해주세요.

### Q. [답변 완료]를 눌렀는데 멈췄어요.
*   일시적인 네트워크 지연이나 AI 응답이 늦어질 수 있습니다. 
*   30분 이상 지연된다면 서버 콘솔(`server.py` 실행 창)에 에러 메시지가 있는지 확인해주세요.

### Q. 카메라가 안 나와요.
*   다른 프로그램(Zoom 등)에서 카메라를 사용 중인지 확인하고 종료해주세요.
*   브라우저 주소창 왼쪽의 자물쇠 아이콘을 눌러 카메라 권한을 '허용'으로 설정하세요.
