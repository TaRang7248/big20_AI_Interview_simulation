# 🤖 AI 면접 시뮬레이션 가이드북 (GUIDEBOOK)

본 문서는 AI 면접 시뮬레이션 프로그램의 개요, 환경 설정, 실행 방법 및 주요 기능에 대해 상세히 설명합니다.

---

## 1. 개요 (Overview)
본 프로그램은 사용자(지원자)가 등록된 채용 공고에 지원하고, AI 면접관과 실시간으로 인터뷰를 진행할 수 있는 웹 기반 시뮬레이션 서비스입니다.
- **주요 목적**: 면접 환경 경험 제공 및 역량 평가 시뮬레이션
- **대상**: 취업 준비생(지원자) 및 채용 담당자(관리자)

---

## 2. 개발 및 실행 환경 (Environment)
- **언어**: Python 3.12+ (Backend), JavaScript ES6+ (Frontend)
- **프레임워크**: Flask (Python Web Server)
- **데이터베이스**: PostgreSQL (사용자 및 공고 정보 저장)
- **주요 라이브러리**:
  - `Flask`: 웹 서버 구동
  - `psycopg2-binary`: PostgreSQL 데이터베이스 연결
  - `python-dotenv`: 환경 변수 관리
  - `webbrowser`: 서버 실행 시 브라우저 자동 호출

---

## 3. 프로그램 실행 방법 (How to Run)

### 3.1 사전 준비
1. **PostgreSQL 서비스 실행**: 로컬 시스템에 PostgreSQL이 설치 및 실행 중이어야 합니다.
2. **데이터베이스 생성**: `interview_db`라는 이름의 데이터베이스를 생성합니다.
   ```sql
   CREATE DATABASE interview_db;
   ```

### 3.2 가상환경 활성화 및 패키지 설치
```bash
# 가상환경 활성화 (Windows 기준)
.\interview_env\Scripts\activate

# 필요 라이브러리 설치 (필요 시)
pip install flask psycopg2-binary python-dotenv
```

### 3.3 서버 실행
```bash
python server.py
```
- 서버가 실행되면 자동으로 `http://localhost:5000` 페이지가 브라우저에서 열립니다.

---

## 4. 주요 기능 및 사용법 (Key Features)

### 4.1 로그인 및 회원가입
- **면접자/관리자 구분**: 가입 시 유형을 선택하여 서로 다른 대시보드에 접근합니다.
- **본인 확인**: 개인 정보 수정 시 비밀번호 재확인 절차를 거칩니다.

### 4.2 채용 공고 관리 (관리자 전용)
- **공고 등록/수정/삭제**: 관리자는 새로운 면접 공고를 작성하고 관리할 수 있습니다.

### 4.3 이력서 업로드 및 환경 테스트 (면접자)
- **이력서 제출**: 면접 전 PDF 형식의 이력서를 필수로 업로드해야 합니다.
- **장치 테스트**: 카메라와 마이크 권한을 확인하여 면접 준비 상태를 점검합니다.

### 4.4 실시간 AI 면접
- **단계별 질의**: 자기소개, 역량 검증, 기술 문제 등 3단계로 진행됩니다.
- **자동 타이머**: 각 질문당 60초의 답변 시간이 주어집니다.
- **결과 리포트**: 면접 종료 후 기술/문제해결/소통 등 영역별 점수를 시뮬레이션하여 보여줍니다.
