# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK.md)

본 문서는 웹 기반 AI 면접 시뮬레이션 프로젝트의 개요, 환경 설정, 실행 방법, 주요 기능 및 파일 구조를 설명합니다.

## 1. 시뮬레이션 개요
이 프로젝트는 **웹 기반 AI 면접 시뮬레이션**으로, 사용자가 실제 면접과 유사한 환경에서 AI 면접관과 음성으로 대화하며 면접 연습을 할 수 있도록 돕습니다.
- **주요 목표**: 실전 같은 면접 경험 제공, 음성 인식 및 음성 합성 기술 활용, 면접 결과 피드백 제공.
- **특징**: 웹캠 및 마이크를 이용한 실시간 상호작용, AI 면접관의 음성 질문, 자동 면접 기록 및 평가.

## 2. 시뮬레이션 환경
- **운영체제**: Windows (권장), macOS, Linux
- **Python 버전**: Python 3.10 권장
- **브라우저**: Chrome (최적화), Edge, Firefox 등 최신 브라우저

## 3. 시뮬레이션 실행 방법

### 3.1. 사전 준비
1. **Python 설치**: Python 3.10 버전을 설치합니다.
2. **필수 라이브러리 설치**:
   ```bash
   pip install -r requirements.txt
   ```
   *(참고: PyTorch 등 일부 패키지는 시스템 환경에 따라 별도 설치가 필요할 수 있습니다.)*

### 3.2. 서버 실행
1. 프로젝트 루트 디렉토리(`C:\big20\big20_AI_Interview_simulation\LDW\text09`)에서 터미널을 엽니다.
2. 다음 명령어로 서버를 실행합니다:
   ```bash
   python server.py
   ```
3. 서버가 시작되면 자동으로 기본 웹 브라우저가 열리며 `http://localhost:5000`으로 접속됩니다.
   (자동으로 열리지 않을 경우 브라우저 주소창에 직접 입력하세요.)

## 4. 사용 모델 및 라이브러리 목록
본 시뮬레이션은 다음과 같은 기술 스택을 활용합니다.

### 백엔드 (Backend)
- **FastAPI**: 고성능 비동기 웹 프레임워크
- **Uvicorn**: ASGI 서버
- **SQLAlchemy / Alembic**: 데이터베이스 ORM 및 마이그레이션
- **Pydantic**: 데이터 검증

### AI & 데이터 처리
- **OpenAI / LangChain**: LLM 기반 면접 질문 생성 및 평가 (오케스트레이션)
- **Edge-TTS**: AI 면접관 음성 합성 (TTS)
- **Deepgram / PyAudio**: 음성 인식 (STT) 및 오디오 처리
- **Hume AI**: (선택적) 감정 인식 및 음성 분석
- **DeepFace / OpenCV**: 비언어적(표정, 태도) 분석

### 프론트엔드 (Frontend)
- **Vanilla JS (ES6+)**: 별도 프레임워크 없는 순수 자바스크립트 구현
- **Web Speech API**: 브라우저 내장 음성 인식 (보조)
- **MediaRecorder API**: 사용자 답변 녹음

## 5. 주요 기능 사용법

### 1) 회원가입 및 로그인
- **회원가입**: 아이디, 비밀번호, 이름 등 기본 정보를 입력하여 계정을 생성합니다. (면접자/관리자 선택 가능)
- **로그인**: 생성한 계정으로 접속합니다.

### 2) 면접 공고 확인 및 지원
- **대시보드**: 현재 진행 중인 면접 공고 목록을 확인합니다.
- **상세 보기**: 공고 내용을 확인하고 '지원하기' 버튼을 누릅니다.

### 3) 면접 진행 (핵심 기능)
- **환경 점검**: 카메라와 마이크 상태를 확인합니다.
- **질문 청취**: AI 면접관이 음성으로 질문을 합니다.
- **음성 답변**: 질문이 끝나면 **즉시 녹음이 시작**됩니다.
- **답변 제출**: 답변을 완료하면 '답변 제출' 버튼을 누릅니다.
    - *참고: 답변 제출 버튼은 **녹음 시작 후 10초 뒤에 활성화**됩니다.*
- **자동 진행**: 제출 후 다음 질문으로 자동 연결됩니다.

### 4) 결과 확인
- **마이페이지**: '내 면접 기록'에서 과거 면접 결과(합격/불합격)와 상세 피드백(기술, 인성 등)을 확인할 수 있습니다.

### 5) 관리자 기능
- **공고 관리**: 면접 공고를 등록, 수정, 삭제할 수 있습니다.
- **지원자 관리**: 지원자 현황 및 상세 평가 결과를 조회할 수 있습니다.
