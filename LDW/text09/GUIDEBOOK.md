# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK.md)

## 1. 개요 (Overview)
본 프로젝트는 **웹 기반 AI 면접 시뮬레이션** 프로그램입니다. 지원자는 실제 면접과 유사한 환경에서 AI 면접관의 질문에 답변하고, 실시간으로 피드백을 받을 수 있습니다. 관리자는 채용 공고를 관리하고 지원자의 면접 결과를 확인할 수 있습니다.

### 주요 특징
- **AI 기반 질문 생성**: 직무와 이력서를 분석하여 맞춤형 면접 질문 제공
- **음성 인식/답변**: 실제 면접처럼 말하는 답변 방식 지원 (STT/TTS 시뮬레이션)
- **자동 평가 시스템**: 답변 내용, 길이, 태도 등을 종합적으로 분석하여 점수화
- **채용 공고 관리**: 관리자 페이지를 통한 공고 등록, 수정, 삭제 기능

---

## 2. 환경 설정 (Environment Setup)
본 프로그램을 실행하기 위해 다음 환경이 필요합니다.

### 필수 요구 사항
- **OS**: Windows 10/11 (권장)
- **Python**: 3.8 이상
- **PostgreSQL**: 13 이상 (데이터베이스)
- **Chrome Browser**: 최신 버전

### 설치 필요한 라이브러리
다음 명령어로 필요한 Python 패키지를 설치해주세요.
```bash
pip install flask psycopg2-binary python-dotenv pypdf
```

---

## 3. 프로그램 실행 방법 (How to Run)

1. **데이터베이스 설정**
   - PostgreSQL을 실행하고 `interview_db` 데이터베이스를 생성합니다.
   - `.env` 파일 또는 `server.py`의 `DB_PASS` 변수에 DB 비밀번호를 설정합니다. (기본값: `013579`)

2. **서버 실행**
   - 터미널(CMD/PowerShell)에서 프로젝트 폴더로 이동합니다.
   - 다음 명령어를 실행합니다.
     ```bash
     python server.py
     ```
   - 서버가 정상적으로 실행되면 브라우저가 자동으로 열리며 `http://localhost:5000`으로 접속됩니다.

## 4. 파일 구조 (File Structure)
```
/
├── server.py             # Python Flask 백엔드 서버 (API 및 DB 처리)
├── app.js                # Frontend 로직 (UI 제어, API 호출, 면접 진행)
├── index.html            # 메인 웹 페이지 (SPA 구조)
├── styles.css            # 스타일 시트
├── create_table.py       # (유틸) 테이블 초기화 스크립트 (server.py 내장됨)
├── db_data/              # 데이터베이스 관련 파일
└── uploads/              # 업로드된 이력서 파일 저장소
```

## 5. 주요 기능 사용법 (User Manual)

### [지원자]
1. **회원가입/로그인**: 계정을 생성하고 로그인합니다.
2. **공고 확인**: '지원 가능한 공고' 목록에서 원하는 직무를 선택합니다.
3. **면접 준비**: 이력서(PDF)를 업로드하고 카메라/마이크 테스트를 진행합니다.
4. **실전 면접**: AI 면접관의 질문에 답변합니다. (총 5문항 내외)
5. **결과 확인**: 면접 종료 후 종합 점수와 피드백을 확인합니다.

### [관리자]
1. **관리자 로그인**: 관리자 계정으로 로그인합니다.
2. **공고 관리**: 새로운 채용 공고를 등록하거나 기존 공고를 수정/삭제합니다.
3. **지원자 현황**: 지원자들의 면접 결과 및 점수를 조회합니다.
