# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK)

본 문서는 웹 기반 AI 면접 시뮬레이션 프로그램의 개요, 설치 환경, 실행 방법 및 주요 기능에 대해 설명합니다.

## 1. 프로젝트 개요
이 프로그램은 지원자가 웹 환경에서 AI 면접관과 실시간으로 대화하며 면접을 진행할 수 있는 시뮬레이션 서비스입니다. 지원자의 이력서를 분석하여 맞춤형 질문을 생성하고, 답변을 분석하여 최종 합불 결과와 세부 평가를 제공합니다.

## 2. 개발 환경 및 기술 스택
- **Backend**: Python 3.x, FastAPI
- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Database**: PostgreSQL
- **AI Models**: 
    - **OpenAI GPT-4o**: 질문 생성, 답변 평가, 면접 결과 분석
    - **OpenAI Whisper-1**: 음성 인식 (STT)

## 3. 프로그램 실행 방법
1. **데이터베이스 설정**: PostgreSQL에 `interview_db` 데이터베이스를 생성합니다.
2. **환경 변수 설정**: 프로젝트 루트의 `.env` 파일에 `DB_HOST`, `DB_NAME`, `DB_USER`, `POSTGRES_PASSWORD`, `OPENAI_API_KEY` 등을 설정합니다.
3. **의존성 설치**:
   ```bash
   pip install -r requirements.txt
   ```
4. **테이블 생성 및 마이그레이션**:
   ```bash
   python create_table.py
   python migrate_db_v1~v6.py  # 순차적 실행 또는 최신 버전 실행
   ```
5. **서버 실행**:
   ```bash
   python server.py
   ```
6. **브라우저 접속**: `http://localhost:5000`으로 접속합니다.

## 4. 주요 기능 사용법

### 지원자 메뉴
- **지원 공고 목록**: 현재 진행 중인 채용 공고를 확인하고 지원할 수 있습니다.
- **면접 시작**: 이력서(PDF)를 업로드하고 카메라/마이크 테스트 후 AI 면접을 시작합니다.
- **내 면접 기록**: 자신이 진행한 모든 면접의 공고 제목, 직무, 시간, 합불 결과를 확인할 수 있습니다.
- **내 정보 관리**: 개인 정보를 수정하거나 비밀번호를 변경할 수 있습니다.

### 관리자 메뉴
- **공고 관리**: 새로운 면접 공고를 등록하거나 수정/삭제할 수 있습니다.
- **지원자 현황**: 지원자들의 면접 진행 상황과 결과를 모니터링할 수 있습니다.

## 5. 파일 구조 설명
- `server.py`: FastAPI 서버 및 API 엔드포인트 로직
- `index.html`: 메인 웹 페이지 구조
- `app.js`: 프론티엔드 라우팅 및 클라이언트 로직
- `styles.css`: 웹 스타일링 정의
- `db/`: 데이터베이스 관련 폴더
- `uploads/`: 업로드된 이력서 및 오디오 파일 저장소
- `migrate_db_*.py`: 데이터베이스 스키마 변경을 위한 마이그레이션 스크립트
- `GUIDEBOOK.md`: 현재 보고 계신 프로젝트 가이드 문서

## 6. 주의 사항
- 면접 진행 시 마이크와 카메라 권한 허용이 필수적입니다.
- OpenAI API 호출을 위해 인터넷 연결이 필요합니다.
