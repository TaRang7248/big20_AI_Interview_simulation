# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK)

## 1. 개요 (Overview)
본 프로그램은 **웹 기반 AI 면접 시뮬레이션**입니다. 사용자는 면접관(AI)과 상호작용하며 실제 면접과 유사한 환경에서 연습할 수 있습니다. 관리자는 면접 공고를 등록, 수정, 삭제하고 지원자들의 면접 결과를 확인할 수 있습니다.

### 주요 기능
- **지원자 모드**:
    - 회원가입 및 로그인
    - 채용 공고 조회 및 지원
    - 웹캠/마이크 환경 테스트
    - AI 면접 진행 (TTS로 질문 듣기, STT로 답변 하기)
    - 면접 결과 확인 (점수 및 합불 예측)
- **관리자 모드**:
    - 공고 관리 (등록, 수정, 삭제, 조회)
    - 지원자 현황 파악 (합격/불합격 여부, 상세 점수)

## 2. 환경 설정 (Environment Setup)
본 프로그램은 **Python (Flask)** 백엔드와 **Vanilla JS** 프론트엔드로 구성되어 있으며, **PostgreSQL** 데이터베이스를 사용합니다.

### 필수 요구 사항
- **OS**: Windows (권장), macOS, Linux
- **Python**: 3.8 이상
- **Database**: PostgreSQL (Docker 컨테이너 또는 로컬 설치)
- **Browser**: Chrome, Edge 등 모던 브라우저 (카메라/마이크 권한 필요)

### 설치 방법
1. 소스 코드를 다운로드합니다.
2. `requirements.txt`에 명시된 Python 패키지를 설치합니다 (필요 시).
   ```bash
   pip install flask psycopg2-binary python-dotenv
   ```
3. PostgreSQL 데이터베이스가 실행 중이어야 하며, `.env` 파일 또는 `server.py`에 DB 접속 정보가 설정되어 있어야 합니다.

## 3. 프로그램 실행 방법 (Execution)
터미널(CMD, PowerShell 등)에서 `server.py`가 있는 디렉토리로 이동한 후 아래 명령어를 실행합니다.

```bash
python server.py
```

- 서버가 실행되면 브라우저가 자동으로 열리며 `http://localhost:5000`으로 접속됩니다.
- 자동으로 열리지 않을 경우 수동으로 주소를 입력하여 접속하세요.

## 4. 사용 라이브러리 (Libraries)
### Backend (Python)
- **Flask**: 웹 서버 프레임워크
- **psycopg2**: PostgreSQL 데이터베이스 연동
- **python-dotenv**: 환경 변수 관리

### Frontend (HTML/CSS/JS)
- **Vanilla JS**: 별도 프레임워크 없이 순수 자바스크립트로 SPA(Single Page Application) 구조 구현
- **Web APIs**:
    - `getUserMedia`: 웹캠 및 마이크 접근
    - `SpeechSynthesis` (Web Speech API): TTS (텍스트 음성 변환) - AI 면접관 음성
    - `SpeechRecognition` (Web Speech API): STT (음성 텍스트 변환) - 사용자 답변 인식

## 5. 주요 기능 사용법 (User Guide)

### [공통] 회원가입 및 로그인
1. 메인 화면에서 '회원가입' 링크를 클릭합니다.
2. 아이디, 비밀번호, 이름, 생년월일, 연락처 등을 입력합니다.
3. **회원 유형**을 선택합니다.
    - **면접자**: 일반 지원자용
    - **관리자**: 공고 및 지원자 관리용
4. 가입 후 로그인 화면에서 아이디/비밀번호로 로그인합니다.

### [관리자] 공고 관리
1. 관리자 계정으로 로그인합니다.
2. 좌측 메뉴에서 **'공고 관리'**를 클릭합니다.
3. **공고 등록**: 우측 상단 '공고 등록' 버튼을 눌러 제목과 마감일을 입력하고 등록합니다.
4. **공고 수정**: 목록에서 '수정' 버튼을 눌러 내용을 변경합니다.
5. **공고 삭제**: 목록에서 '삭제' 버튼을 눌러 공고를 삭제합니다. (삭제 시 복구 불가)

### [지원자] 면접 진행
1. 지원자 계정으로 로그인합니다.
2. **'지원 공고 목록'**에서 원하는 공고의 '지원하기' 버튼을 클릭합니다.
3. **환경 테스트**: 카메라와 마이크 권한을 허용하고 작동 여부를 확인합니다.
4. **면접 시작**: '면접 시작' 버튼을 누르면 면접이 진행됩니다.
    - AI 면접관이 질문을 읽어줍니다.
    - 답변 시간이 주어지며, 마이크를 통해 답변하거나 텍스트로 수정할 수 있습니다.
5. **결과 확인**: 면접이 종료되면 바로 결과를 확인할 수 있습니다.

```
├── styles.css          # 스타일 시트

```
