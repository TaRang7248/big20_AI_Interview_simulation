# AI 면접 시뮬레이션 웹 애플리케이션 가이드북

## 1. 개요
본 프로젝트는 웹 기반의 **AI 면접 시뮬레이션 프로그램**입니다. 사용자는 지원자(Applicant) 또는 관리자(Admin) 권한으로 로그인하여 면접 공고 확인, 면접 진행, 결과 확인 및 공고 관리 기능을 이용할 수 있습니다.

## 2. 개발 환경
- **Language**: Python 3.x, JavaScript (ES6+)
- **Framework**: Flask (Backend), Vanilla JS (Frontend)
- **Database**: PostgreSQL (Connects via `psycopg2`)
- **Frontend**: HTML5, CSS3, JavaScript (Single Page Application 구조)

## 3. 프로그램 실행 방법

### 3.1. 필수 라이브러리 설치
터미널에서 다음 명령어를 입력하여 필요한 라이브러리를 설치합니다.
```bash
pip install flask psycopg2-binary python-dotenv
```

### 3.2. 서버 실행
프로젝트 디렉토리(`C:\big20\big20_AI_Interview_simulation\LDW\text09`)로 이동한 후 서버를 실행합니다.
```bash
python server.py
```
서버가 정상적으로 실행되면 `http://0.0.0.0:5000` 주소로 호스팅됩니다.

### 3.3. 접속
웹 브라우저를 열고 `http://localhost:5000`에 접속합니다.

## 4. 사용 라이브러리
- **Flask**: 웹 서버 프레임워크
- **psycopg2**: PostgreSQL 데이터베이스 연동 및 관리
- **python-dotenv**: 환경 변수(.env) 로드
- **webbrowser**: 서버 실행 시 브라우저 자동 오픈 (선택적)

## 5. 주요 기능 사용법

### 5.1. 회원가입 및 로그인
- **회원가입**: ID(아이디), PW(비밀번호), 이름, 생년월일, 성별, 이메일, 주소, 전화번호를 입력하여 가입합니다. '지원자' 또는 '관리자' 유형을 선택할 수 있습니다.
- **로그인**: 가입한 아이디와 비밀번호로 로그인합니다.

### 5.2. 지원자 (Applicant) 기능
- **공고 확인**: 등록된 면접 공고 리스트를 확인하고 '지원하기' 버튼을 클릭합니다.
- **사전 점검**: 웹캠 및 마이크 작동 여부를 테스트합니다.
- **면접 진행**:
  - **자기소개 -> 역량 검증 -> 기술 면접** 단계로 진행됩니다.
  - AI 질문에 대해 음성(또는 텍스트)으로 답변합니다.
  - 타이머가 작동하며 제한 시간 내에 답변해야 합니다.
- **결과 확인**: 면접 종료 후 AI가 분석한 점수(기술, 문제해결, 의사소통, 태도)와 합격 여부 예측 결과를 확인합니다.
- **내 정보 수정**: 비밀번호 확인 과정을 거친 후 이메일, 주소, 전화번호를 수정하거나 비밀번호를 변경할 수 있습니다.

### 5.3. 관리자 (Admin) 기능
- **공고 관리**: 새로운 채용 공고를 등록, 수정, 삭제할 수 있습니다.
  - 공고 목록에는 제목, 작성자(ID), 작성일, 마감일이 표시됩니다.
- **지원자 관리**: 면접을 진행한 지원자들의 기록과 점수를 조회할 수 있습니다. (Mock Data 기반 데모 기능 포함)

## 6. 파일 구조 설명

| 파일명 | 설명 |
|---|---|
| `server.py` | Flask 기반 백엔드 서버 코드. API 엔드포인트 정의 및 DB 연동 처리. |
| `app.js` | 프론트엔드 로직. SPA 라우팅, API 호출, 면접 프로세스 제어 등 포함. |
| `index.html` | 웹 애플리케이션의 메인 진입점. UI 레이아웃 정의. |
| `styles.css` | 애플리케이션의 스타일 디자인 (CSS). |
| `GUIDEBOOK.md` | 프로젝트 사용 가이드북 (본 문서). |
| `migrate_db.py` | (선택) 별도의 DB 마이그레이션 스크립트. |
| `.env` | (상위 디렉토리) 데이터베이스 접속 정보 등 환경 변수 저장. |
