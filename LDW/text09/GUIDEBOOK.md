# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK)

## 1. 개요 (Overview)
본 프로그램은 **웹 기반 AI 면접 시뮬레이션**입니다. 사용자는 면접자로서 가상의 AI 면접관과 음성 및 화상으로 상호작용하며 실전 같은 면접을 경험할 수 있습니다. 관리자는 면접 공고를 관리하고 지원자의 결과를 확인할 수 있습니다.

### 주요 특징
- **실시간 화상/음성 인터랙션**: 웹캠과 마이크를 사용하여 실제 면접 환경을 모사합니다.
- **단계별 면접 진행**: 자기소개, 역량 검증, 기술 면접(코딩 테스트 포함) 등 체계적인 면접 프로세스를 제공합니다.
- **AI 면접관 (TTS/STT)**: AI가 질문을 음성으로 읽어주고(TTS), 사용자의 답변을 텍스트로 변환(STT)하여 기록합니다.
- **회원 관리 및 대시보드**: 지원자와 관리자 계정을 분리하여 각각 맞춤형 대시보드를 제공합니다.

## 3. 프로그램 실행 방법 (How to Run)

### 1단계: 필수 라이브러리 설치
본 프로그램은 Python 내장 라이브러리를 주로 사용하므로, 별도의 복잡한 설치 과정은 없으나 `Flask` 설치가 필요합니다. 터미널(CMD 또는 PowerShell)에서 다음 명령어를 실행하세요.

```bash
pip install flask
```

### 2단계: 서버 실행
소스 코드가 있는 디렉토리(`C:\big20\big20_AI_Interview_simulation\LDW\text09`)로 이동한 후, 다음 명령어를 입력하여 서버를 시작합니다.

```bash
python server.py
```

### 3단계: 접속
서버가 실행되면 웹 브라우저가 자동으로 열립니다. 만약 열리지 않는다면 브라우저 주소창에 아래 주소를 입력하세요.

- **접속 주소**: [http://localhost:5000](http://localhost:5000)

## 4. 사용 라이브러리 (Libraries)

### Backend (Python)
- **Flask**: 경량 웹 프레임워크로, 정적 파일 서빙 및 API 엔드포인트를 처리합니다.
- **sqlite3**: 내장 데이터베이스 라이브러리로, 사용자 정보 및 면접 데이터를 저장합니다. (파일 기반 DB: `db/membership_information.db`)
- **webbrowser**: 서버 실행 시 자동으로 브라우저를 열어줍니다.
- **threading**: 타이머 기능을 위해 사용됩니다.
- **os**: 파일 시스템 경로 처리 등을 위해 사용됩니다.

### Frontend (Web)
- **Vanilla JavaScript (ES6+)**: 별도의 프레임워크(React, Vue 등) 없이 순수 자바스크립트로 SPA(Single Page Application)를 구현했습니다.
- **HTML5/CSS3**: 시멘틱 마크업과 반응형 스타일링을 적용했습니다.
- **Web APIs**:
    - `getUserMedia`: 카메라 및 마이크 접근.
    - `SpeechRecognition (Web Speech API)`: 음성 인식(STT).
    - `SpeechSynthesis (Web Speech API)`: 음성 합성(TTS).

## 5. 주요 기능 사용법 (User Manual)

### [1] 회원가입 및 로그인
1. 초기 화면에서 **회원가입** 링크를 클릭합니다.
2. 아이디, 비밀번호, 이름, 정보 등을 입력하고 **가입하기**를 누릅니다.
    - *면접자* 또는 *관리자* 유형을 선택할 수 있습니다.
3. 가입한 정보로 **로그인**합니다.

### [2] 면접 지원 (면접자)
1. 로그인 후 **지원 가능한 공고** 목록에서 원하는 공고의 **지원하기** 버튼을 클릭합니다.
2. **이력서 업로드** (시뮬레이션) 및 **환경 테스트**를 진행합니다.
    - *카메라/마이크* 버튼을 눌러 권한을 허용해야 면접 시작 버튼이 활성화됩니다.
3. **면접 시작** 버튼을 누릅니다.

### [3] 실전 면접 진행
1. **AI 면접관**이 질문을 음성으로 읽어줍니다.
2. 질문이 끝나면 **타이머**가 작동하며 마이크가 켜집니다.
3. 답변을 말하면 실시간으로 텍스트로 변환되어 입력창에 표시됩니다.
4. **답변 완료** 버튼을 누르거나 시간이 종료되면 다음 질문으로 넘어갑니다.
5. 모든 단계(자기소개 -> 역량 -> 기술)가 끝나면 결과 페이지로 이동합니다.

### [4] 회원 정보 수정
1. 대시보드 사이드바 메뉴에서 **내 정보**를 클릭합니다.
2. 비밀번호 확인 후 이메일이나 전화번호를 수정할 수 있습니다.
3. 수정을 원치 않을 경우 새로 추가된 **취소** 버튼을 눌러 대시보드로 돌아갑니다.
