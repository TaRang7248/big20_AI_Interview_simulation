# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK)

## 1. 개요 (Overview)
본 프로그램은 사용자가 AI와 모의 면접을 진행할 수 있는 웹 기반 시뮬레이션입니다. 사용자는 회원가입 및 로그인을 통해 이력서를 등록하고, 실제 면접처럼 카메라와 마이크를 사용하여 질문에 답변할 수 있습니다. 면접 후에는 AI가 분석한 결과를 확인할 수 있습니다.

## 2. 환경 (Environment)
- **OS**: Windows (권장)
- **Language**: Python 3.8+, JavaScript (ES6+)
- **Backend Framework**: Flask
- **Frontend**: HTML5, CSS3, Vanilla JS
- **Database**: SQLite

## 3. 프로그램 실행 방법 (How to Run)
1. `server.py`가 있는 디렉토리로 이동합니다.
2. 터미널(CMD 또는 PowerShell)에서 다음 명령어를 실행합니다.
   ```bash
   python server.py
   ```
3. 프로그램이 실행되면 자동으로 브라우저가 열리며 `http://localhost:5000`으로 접속됩니다.
4. 만약 브라우저가 열리지 않으면 수동으로 주소창에 입력하여 접속하세요.

## 4. 사용 라이브러리 (Libraries)
- **Flask**: 웹 서버 구축을 위한 파이썬 마이크로 프레임워크
- **sqlite3**: 내장 데이터베이스 (별도 설치 불필요)
- **webbrowser**: 웹 브라우저 자동 실행
- **threading**: 타이머 기능 사용

## 5. 주요 기능 사용법 (Features)

### 5.1 회원가입 및 로그인
- 계정이 없다면 '회원가입'을 통해 ID, 비밀번호, 이름 등을 등록합니다.
- 등록된 정보로 로그인하여 대시보드에 접근합니다.

### 5.2 내 정보 수정 (비밀번호 확인 포함)
- 대시보드에서 '내 정보' 메뉴를 클릭합니다.
- **본인 확인을 위해 비밀번호를 입력합니다.** (신규 기능)
- 비밀번호가 확인되면 이메일과 전화번호를 수정할 수 있는 페이지로 이동합니다.
- '정보 수정 완료' 버튼을 누르면 **자동으로 메인 페이지로 이동**합니다. (신규 기능)

### 5.3 면접 진행
- 공고 목록에서 원하는 직무의 '지원하기' 버튼을 누릅니다.
- 카메라와 마이크 권한을 허용하고 장치 테스트를 진행합니다.
- '면접 시작'을 누르면 AI 면접관과 질의응답이 시작됩니다.
- 각 질문에 대해 음성으로 답변하거나, 텍스트 입력을 사용할 수 있습니다.

### 5.4 결과 확인
- 면접이 종료되면 기술, 문제 해결, 의사소통, 태도 영역별 점수를 확인할 수 있습니다.
- 합격/불합격 예측 결과를 볼 수 있습니다.
