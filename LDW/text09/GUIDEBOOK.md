# AI 면접 시뮬레이션 가이드북 (AI Interview Simulation Guidebook)

## 1. 개요 (Overview)
본 프로젝트는 웹 기반의 **AI 면접 시뮬레이션 프로그램**입니다. 사용자는 면접자로서 모의 면접을 진행하고, 관리자는 공고를 등록 및 관리할 수 있습니다. Python Flask를 백엔드로, HTML/CSS/Vanilla JS를 프론트엔드로 사용하여 가볍고 빠른 웹 애플리케이션을 구현했습니다. 데이터베이스는 PostgreSQL을 사용하여 사용자 정보와 채용 공고를 관리합니다.

## 2. 개발 환경 (Environment)
본 프로젝트는 다음과 같은 환경에서 개발 및 실행 테스트되었습니다.
- **OS**: Windows / Linux / macOS (Cross-platform 지원)
- **Language**: Python 3.8+, JavaScript (ES6+)
- **Backend Framework**: Flask
- **Database**: PostgreSQL
- **IDE**: VS Code (권장)

## 3. 프로그램 실행 방법 (How to Run)

### 3.1. 사전 준비
1. Python 설치 확인 (`python --version`)
2. PostgreSQL 설치 및 실행 확인
3. 필요한 패키지 설치:
   ```bash
   pip install flask psycopg2-binary python-dotenv
   ```
4. `.env` 파일 설정 (프로젝트 루트 경로에 위치, DB 접속 정보 포함)

### 3.2. 서버 실행
터미널에서 `server.py`가 있는 디렉토리로 이동 후 아래 명령어를 실행합니다.
```bash
python server.py
```
- 서버가 정상적으로 실행되면 브라우저가 자동으로 열리거나 `http://localhost:5000`으로 접속하여 확인할 수 있습니다.
- 초기 실행 시 DB 테이블이 자동으로 생성됩니다.

## 4. 사용 라이브러리 (Libraries)
- **Flask**: 웹 서버 구축 및 API 라우팅 처리
- **psycopg2**: PostgreSQL 데이터베이스 연동
- **python-dotenv**: 환경 변수 관리 (.env 파일 로드)
- **webbrowser**: 서버 실행 시 브라우저 자동 실행

## 5. 주요 기능 사용법 (Features)

### 5.1. 회원가입 및 로그인
- **회원가입**: 아이디, 비밀번호, 이름, 생년월일, 성별, 이메일, 주소, 전화번호를 입력하여 가입합니다. '면접자' 또는 '관리자' 유형을 선택할 수 있습니다.
- **로그인**: 가입한 아이디와 비밀번호로 로그인합니다.

### 5.2. [면접자] 면접 진행
- **대시보드**: 지원 가능한 공고 목록을 확인하고 '지원하기' 버튼을 클릭합니다.
- **환경 설정**: 카메라 및 마이크 테스트, 이력서 업로드(시뮬레이션)를 진행합니다.
- **면접 진행**: AI 면접관의 질문(음성 안내 및 텍스트)에 따라 답변을 입력하거나 말합니다. 제한 시간이 주어지며 단계별로(자기소개, 역량 검증, 기술 면접) 진행됩니다.
- **결과 확인**: 면접 종료 후 기술, 문제 해결, 의사소통, 태도 등 영역별 점수와 합격 여부 예측 결과를 확인합니다.

### 5.3. [관리자] 공고 관리
- **공고 목록**: 등록된 모든 면접 공고를 리스트로 확인합니다.
- **공고 등록**: 제목, 내용, 마감일을 입력하여 새로운 채용 공고를 등록합니다. 입사 지원자들에게 즉시 노출됩니다.
- **공고 수정/삭제**: 기존 공고 내용을 수정하거나 삭제할 수 있습니다.

### 5.4. 마이페이지
- **정보 수정**: 자신의 정보를 조회하고 비밀번호 검증 후 수정할 수 있습니다.
- **비밀번호 변경**: 현재 비밀번호 확인 후 새로운 비밀번호로 변경 가능합니다.

## 6. 파일 구조 설명 (File Structure)

```
/
├── server.py              # 메인 백엔드 서버 파일 (Flask 앱, DB 연결, API 엔드포인트)
├── index.html             # 메인 프론트엔드 HTML (SPA 구조)
├── styles.css             # 스타일 시트 (UI/UX 디자인)
├── app.js                 # 프론트엔드 로직 (SPA 라우팅, API 호출, 인터랙션)
├── db/                    # 데이터베이스 관련 파일 (백업 등)
├── check_table.py         # DB 테이블 상태 확인 스크립트
├── create_table.py        # DB 초기 테이블 생성 스크립트
├── diagnose_db.py         # DB 연결 진단 도구
├── migrate_db.py          # DB 스키마 마이그레이션 도구
└── GUIDEBOOK.md           # 프로젝트 가이드북 (본 파일)
```
