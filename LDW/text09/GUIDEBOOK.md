# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK)

## 1. 개요 (Overview)
본 프로그램은 **웹 기반 AI 면접 시뮬레이션 시스템**입니다. 사용자는 이력서를 등록하고, 생성형 AI(OpenAI GPT-4o)가 생성한 맞춤형 질문에 대해 음성으로 답변하며 실제 면접과 유사한 환경을 경험할 수 있습니다. 면접 종료 후 AI가 답변을 분석하여 피드백과 점수를 제공합니다.

## 2. 환경 설정 (Environment)
본 프로그램은 다음 환경에서 구동됩니다.

- **OS**: Windows (테스트 환경)
- **Language**: Python 3.10+, JavaScript (ES6+)
- **Database**: PostgreSQL
- **Browser**: Chrome, Edge 등 최신 브라우저 (카메라/마이크 권한 필요)

### 필수 라이브러리
`requirements.txt`에 명시된 주요 라이브러리는 다음과 같습니다.
- `fastapi`, `uvicorn`: 웹 서버 및 API 프레임워크
- `psycopg2-binary`: PostgreSQL 데이터베이스 연동
- `openai`: AI 면접 질문 생성 및 답변 분석
- `pypdf`: 이력서(PDF) 텍스트 추출
- `python-multipart`: 파일 업로드 처리

## 3. 프로그램 실행 방법 (How to Run)

1. **데이터베이스 실행**: PostgreSQL 서버가 실행 중이어야 하며, `.env` 파일에 DB 접속 정보가 설정되어 있어야 합니다.
2. **서버 실행**:
   터미널에서 프로젝트 폴더로 이동 후 아래 명령어를 실행합니다.
   ```bash
   python server.py
   ```
   서버가 정상적으로 실행되면 브라우저가 자동으로 열리거나 `http://localhost:5000`으로 접속할 수 있습니다.

## 4. 주요 기능 사용법 (Usage)

### 4.1 회원가입 및 로그인
- **회원가입**: ID, 비밀번호, 이름 등 필수 정보를 입력하여 가입합니다.
  - **중복 확인**: 가입 시 ID 중복 여부를 즉시 확인할 수 있습니다.
- **로그인**: 가입한 계정으로 로그인합니다. (관리자/면접자 유형 구분)

### 4.2 면접 준비 (지원자)
1. **공고 확인**: 대시보드에서 지원 가능한 면접 공고를 확인하고 '지원하기'를 클릭합니다.
2. **이력서 업로드**: PDF 형식의 이력서를 업로드합니다.
3. **환경 테스트**: 카메라와 마이크 작동 여부를 확인합니다.

### 4.3 AI 면접 진행
- **질문 청취**: AI 면접관이 직무에 맞는 질문을 음성(TTS)과 텍스트로 제시합니다.
- **답변 하기**: 사용자의 답변이 음성으로 녹음되며, 실시간으로 텍스트로 변환(STT)됩니다.
- **질문 단계**: 자기소개 -> 직무/기술 질문 -> 인성 질문 -> 마무리 순으로 진행됩니다.

### 4.4 결과 분석
- 면접이 종료되면 AI가 답변 내용, 태도 등을 종합적으로 분석하여 평가 리포트를 생성합니다.
- 기술, 문제해결, 의사소통, 비언어적 요소에 대한 점수와 피드백을 확인할 수 있습니다.

### 4.5 관리자 기능
- 관리자 계정으로 로그인 시, 공고를 등록/수정/삭제할 수 있으며 지원자들의 면접 결과를 조회할 수 있습니다.

## 5. 파일 구조 설명 (File Structure)

| 파일/폴더 | 설명 |
| :--- | :--- |
| `server.py` | FastAPI 기반 백엔드 서버. API 엔드포인트 및 비즈니스 로직(DB, AI 연동) 처리 |
| `app.js` | 프론트엔드 로직. SPA 라우팅, API 호출, 미디어 제어(녹음, 카메라) 담당 |
| `index.html` | 메인 웹 페이지 구조 (HTML) |
| `styles.css` | 웹 페이지 스타일 (CSS) |
| `db/` | 데이터베이스 관련 파일 (스키마 등) |
| `uploads/` | 업로드된 이력서 및 녹음 파일 저장소 |
| `verify_*.py` | 기능 검증을 위한 테스트 스크립트 |

## 6. 사용된 모델 및 라이브러리 상세
- **LLM (Language Model)**: GPT-4o (OpenAI) - 면접 질문 생성 및 답변 평가
- **STT (Speech-to-Text)**: Whisper-1 (OpenAI API) & Web Speech API (Real-time) - 음성 인식
- **TTS (Text-to-Speech)**: Web Speech API (Browser Native) - 질문 음성 출력
