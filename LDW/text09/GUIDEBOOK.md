# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK.md)

## 1. 개요 (Overview)
본 프로젝트는 **웹 기반 AI 면접 시뮬레이션** 프로그램입니다. 지원자는 실제 면접과 유사한 환경에서 AI 면접관과 질의응답을 진행하고, 자신의 역량을 평가받을 수 있습니다. 관리자는 채용 공고를 관리하고 지원자의 면접 결과를 확인할 수 있습니다.

### 주요 특징
- **실시간 화상 면접**: 웹캠과 마이크를 활용한 비대면 면접 환경 제공
- **음성 인식/합성**: STT(Speech-to-Text)와 TTS(Text-to-Speech)를 활용한 대화형 면접
- **결과 분석**: 면접 종료 후 기술/문제해결/의사소통/태도 4가지 영역별 점수 제공
- **관리자 모드**: 공고 등록 및 지원자 평가 결과 관리

---

## 2. 환경 설정 (Environment Setup)

### 필수 요구 사항
- **Python 3.8 이상**
- **Google Chrome** (또는 최신 웹 브라우저)
- **웹캠 및 마이크** (면접 체험 시 필수)

### 사용 라이브러리
| 라이브러리 | 용도 | 비고 |
| --- | --- | --- |
| `flask` | 웹 서버 프레임워크 | 백엔드 API 및 정적 파일 서빙 |
| `psycopg2` | PostgreSQL 데이터베이스 연동 | 데이터 저장소 연결 |
| `python-dotenv` | 환경 변수 관리 | DB 접속 정보 등 관리 |
| `webbrowser` | 브라우저 자동 실행 | 서버 시작 시 편의 기능 |

### 설치 방법
프로젝트 루트 디렉토리에서 다음 명령어를 실행하여 필수 패키지를 설치하십시오.
```bash
pip install flask psycopg2 python-dotenv
```
*(참고: `requirements.txt` 파일이 있는 경우 `pip install -r requirements.txt` 명령어로 일괄 설치 가능합니다.)*

## 3. 프로그램 실행 방법 (Execution)

### 서버 실행
터미널에서 `server.py`가 위치한 디렉토리(`C:\big20\big20_AI_Interview_simulation\LDW\text09`)로 이동한 후 아래 명령어를 입력합니다.

```bash
python server.py
```

### 접속
- 서버가 정상적으로 실행되면, 자동으로 기본 웹 브라우저가 열리며 `http://localhost:5000`으로 접속됩니다.
- 만약 자동으로 열리지 않는다면, 브라우저 주소창에 직접 입력하여 접속할 수 있습니다.

## 4. 주요 기능 사용법 (Usage Guide)

### 4.1 회원가입 및 로그인
1. 초기 화면에서 **[회원가입]** 링크를 클릭합니다.
2. 아이디, 비밀번호, 이름, 생년월일, 성별, 이메일, 주소, 전화번호를 입력합니다.
3. **회원 유형**을 선택합니다.
    - **면접자**: 면접 응시 및 결과 확인
    - **관리자**: 공고 관리 및 전체 지원자 현황 조회
4. 가입 완료 후 로그인합니다.

### 4.2 면접자 (Applicant)
- **지원 공고 목록**: 현재 진행 중인 채용 공고를 확인하고 [지원하기] 버튼을 누릅니다.
- **면접 준비**: 이력서를 업로드하고 카메라/마이크 테스트를 진행합니다.
- **실전 면접**:
    - AI 면접관의 질문을 듣고(TTS) 음성으로 답변(STT)합니다.
    - 화면에 남은 시간과 진행 단계가 표시됩니다.
    - 코딩 테스트 단계에서는 코드 입력창이 활성화됩니다.
- **결과 확인**: 면접 종료 후 즉시 영역별 점수와 합격 예측 결과를 확인할 수 있습니다.
- **내 정보 수정**: 사이드바의 [내 정보] 메뉴에서 개인정보를 수정하거나 비밀번호를 변경할 수 있습니다.

### 4.3 관리자 (Admin)
- **공고 관리**: 새로운 채용 공고를 등록하거나 기존 공고를 관리합니다.
- **지원자 현황**: 면접을 완료한 지원자들의 점수와 합격 여부를 목록으로 확인합니다.
- **내 정보 수정**: 관리자 계정의 정보 또한 [내 정보] 메뉴에서 수정 가능합니다.

## 5. 파일 구조 설명 (File Structure)

```
C:\big20\big20_AI_Interview_simulation\LDW\text09
│
├── server.py              # Flask 웹 서버 메인 파일 (API, DB연동)
├── app.js                 # 프론트엔드 로직 (SPA 라우팅, 면접 진행, 이벤트 처리)
├── index.html             # 메인 HTML UI 구조
├── styles.css             # 스타일시트 (CSS)
├── GUIDEBOOK.md           # 프로젝트 가이드북 (본 문서)
│
├── db/                    # 데이터베이스 관련 파일
│   └── membership_information.db (SQLite/PostgreSQL 등 설정에 따름)
│
└── ... (기타 유틸리티 파일)
```

### 주요 파일 역할
- **`server.py`**: 클라이언트의 요청(`login`, `register`, `user info update` 등)을 처리하고 DB와 통신합니다.
- **`app.js`**: `index.html` 내의 화면 전환(`navigateTo`)과 사용자 인터랙션(버튼 클릭, 폼 제출)을 담당합니다. `MOCK_DB`를 통해 일부 데모 데이터를 관리하며, 실제 데이터는 `fetch` API를 통해 서버와 주고받습니다.
- **`styles.css`**: 반응형 디자인과 모던한 UI 스타일을 정의합니다.
