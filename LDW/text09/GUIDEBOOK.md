# AI 면접 시뮬레이션 가이드북 (GUIDEBOOK)

## 1. 개요
본 프로젝트는 **웹 기반 AI 면접 시뮬레이션 프로그램**입니다.
사용자는 지원자로서 이력서를 등록하고, 카메라와 마이크를 통해 AI 면접관과 실시간으로 음성 면접을 진행할 수 있습니다.
면접 종료 후 AI가 분석한 평가 결과(기술, 문제해결, 의사소통, 비언어적 요소)와 합격 여부를 확인할 수 있습니다.

## 2. 개발 환경
- **OS**: Windows
- **Backend**: Python (FastAPI), PostgreSQL
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **AI**: OpenAI API (GPT-4o, Whisper)

## 3. 프로그램 실행 방법

### 1) 환경 설정
필요한 라이브러리를 설치합니다.
```bash
pip install -r requirements.txt
```
*주요 라이브러리: `fastapi`, `uvicorn`, `psycopg2`, `openai`, `python-dotenv`, `pypdf` 등*

### 2) 데이터베이스 설정
PostgreSQL 데이터베이스가 실행 중이어야 합니다.
`.env` 파일에 DB 접속 정보와 OpenAI API Key가 올바르게 설정되어 있는지 확인하세요.

### 3) 서버 실행
터미널에서 `server.py`가 있는 디렉토리로 이동 후 다음 명령어를 실행합니다.
```bash
uvicorn server:app --reload
```
또는 파이썬 스크립트로 직접 실행할 수도 있습니다.
```bash
python server.py
```

### 4) 접속
웹 브라우저를 열고 `http://localhost:8000` (또는 설정된 포트)로 접속합니다.
(기본 `index.html`을 직접 열거나, FastAPI의 정적 파일 호스팅을 이용할 수 있습니다.)

## 4. 주요 기능 사용법

### [지원자]
1. **회원가입/로그인**: 계정을 생성하고 로그인합니다.
2. **공고 확인**: '지원 가능한 공고' 목록에서 원하는 직무를 확인합니다.
3. **내 정보 수정**: 사이드바의 '내 정보'를 통해 개인정보를 확인하고 수정할 수 있습니다.
4. **면접 준비**:
    - 공고의 '확인하기' -> '지원하기' 버튼을 클릭합니다.
    - 이력서(PDF)를 업로드합니다.
    - 카메라와 마이크 권한을 허용하고 연결 상태를 확인합니다. (자동 점검)
5. **실전 면접**:
    - AI 면접관의 질문을 듣고(TTS) 음성으로 답변합니다.
    - 답변이 끝나면 '답변 제출' 버튼을 누르거나, 시간이 종료되면 자동 제출됩니다.
    - 총 5~10개의 질문이 이어집니다.
6. **결과 확인**: 면접 종료 후 잠시 기다리면 합격/불합격 여부와 상세 평가 점수를 확인할 수 있습니다.

### [관리자]
1. **공고 관리**: 관리자 계정으로 로그인하여 채용 공고를 등록, 수정, 삭제할 수 있습니다.
2. **지원자 현황**: (기능 예정) 지원자들의 면접 진행 상황을 모니터링할 수 있습니다.

## 5. 파일 구조 설명
- `server.py`: 백엔드 메인 서버 파일 (API 엔드포인트 정의)
- `index.html`: 웹 애플리케이션의 메인 HTML 구조
- `styles.css`: 전체 스타일시트
- `app.js`: 프론트엔드 로직 (라우팅, API 호출, 면접 진행, 미디어 제어 등)
- `create_table.py`: 데이터베이스 테이블 생성 스크립트
- `requirements.txt`: 의존성 패키지 목록
- `uploads/`: 업로드된 이력서 및 오디오 파일 저장소

## 6. 문제 해결 (FAQ)
- **정보가 보이지 않아요**: '내 정보' 페이지에서 정보가 빈 칸으로 보인다면, 새로고침을 하거나 재로그인 해보세요. (최신 패치로 해결됨)
- **마이크가 안 돼요**: 브라우저의 마이크 권한이 허용되어 있는지 주소창 옆 아이콘을 통해 확인하세요.
