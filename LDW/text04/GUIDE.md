# AI 면접 시뮬레이션 가이드북

본 프로그램은 FastAPI, OpenAI GPT-4o, Whisper STT, 그리고 PostgreSQL(pgvector)을 활용한 지능형 면접 시뮬레이션 시스템입니다.

## 🚀 시작하기

1. **환경 설정**:
   - `.env` 파일에 `OPENAI_API_KEY`와 PostgreSQL 접속 정보를 입력하세요.
   - 프로젝트 경로 (`C:\big20\big20_AI_Interview_simulation\LDW\text04`)에서 터미널을 엽니다.

2. **의존성 설치**:
   ```bash
   pip install -r requirements.txt
   ```
   *주의: `pyaudio` 설치를 위해 시스템에 PortAudio가 필요할 수 있습니다.*

3. **데이터베이스 초기화**:
   - PostgreSQL 서버가 실행 중인지 확인하세요.
   - `pgvector` 확장이 활성화되어 있어야 합니다 (프로그램 실행 시 자동 시도).

4. **프로그램 실행**:
   ```bash
   python main.py
   ```
   *실행과 동시에 브라우저가 자동으로 열립니다 (`http://127.0.0.1:8000`).*

## 🎙️ 면접 진행 방법

1. **정보 입력**: 이름과 지원할 직무(예: 백엔드 개발자)를 입력하고 '면접 시작하기'를 클릭합니다.
2. **질문 확인**: LLM이 `interview.db`와 RAG를 참고하여 첫 질문을 생성합니다.
3. **답변하기**: '답변 시작' 버튼을 누르고 마이크에 대고 말씀하세요.
   - 실시간으로 음성 인식이 진행되는 것을 확인할 수 있습니다.
4. **평가 및 꼬리 질문**: 답변이 완료되면 AI가 이를 분석하여 점수와 피드백을 제공하며, 필요한 경우 날카로운 '꼬리 질문'을 던집니다.
5. **결과 확인**: 모든 질문이 끝나면 최종 면접 로그가 저장됩니다.

## 💾 데이터 저장

- **SQLite (`interview_save.db`)**: 모든 면접 로그(이름, 질문, 답변, 평가)가 실시간으로 저장됩니다.
- **PostgreSQL**: 답변 내용이 벡터 데이터로 저장되어 향후 RAG(검색 증강 생성)에 활용됩니다.

## 💡 팁
- 명확하고 구체적인 답변을 할수록 더 높은 점수와 심도 있는 꼬리 질문을 받을 수 있습니다.
- 주변 소음이 적은 환경에서 마이크 면접을 진행해 주세요.
