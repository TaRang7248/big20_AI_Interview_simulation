# **📝 AI 모의면접 시스템 회의록 (기술스택 정리)**

## **1\. 시스템 준비 단계**

* **회원 로그인 및 정보 입력**  
  * 기술스택: Web Frontend (React, Vue), Backend (Node.js, Django, Spring Boot), DB (PostgreSQL, MySQL, MongoDB)  
  * 기능: 이력서/자소서 업로드, 기업 및 직무 선택  
* **장치 테스트**  
  * 기술스택: WebRTC, MediaPipe, OpenCV  
  * 기능: 카메라/마이크 연결 및 테스트

  ---

  ## **2\. 면접 질문 생성 및 진행**

* **질문 리스트 선정**  
  * 기술스택: LLM (OpenAI GPT, Azure OpenAI, Anthropic Claude), Prompt Engineering  
  * 기능: 기본 질문 리스트 생성  
* **비언어적 키워드 분석 → 꼬리질문 생성**  
  * 기술스택: NLP (spaCy, HuggingFace Transformers), Audio/Video 분석 (OpenCV, PyTorch, TensorFlow)  
  * 기능: 표정, 제스처, 음성 톤 분석 후 꼬리질문 자동 생성  
* **반복 루프**  
  * 기술스택: Orchestration Framework (LangChain, LlamaIndex)  
  * 기능: 질문-답변-분석-꼬리질문 반복

  ---

  ## **3\. 코딩 테스트 및 기술 평가**

* **코딩 테스트 (주관식 문제)**  
  * 기술스택: Online IDE (Monaco Editor, CodeMirror), Sandbox (Docker, Kubernetes)  
  * 기능: 코드 작성 및 실행 환경 제공  
* **LLM 코드 분석 및 검증**  
  * 기술스택: LLM 기반 코드 리뷰 (Codex, GPT-4), Static Analysis Tools (SonarQube, ESLint, Pylint)  
* **화이트보드 평가**  
  * 기술스택: Collaborative Whiteboard (Excalidraw, Miro API), Drawing Recognition (OpenCV, TensorFlow)

  ---

  ## **4\. 라이브 코딩 테스트**

* 기술스택: WebRTC 기반 실시간 협업, Online Judge System (LeetCode-style), Docker Sandbox  
* 기능: 실시간 코딩 및 평가  
  ---

  ## **5\. 평가 및 결과 산출**

* **루브릭 기반 STAR 평가**  
  * 기술스택: Rule-based Scoring Engine, ML 기반 평가 모델  
* **종합 평가 (SWOT 분석 포함)**  
  * 기술스택: LLM Summarization (LangChain, HuggingFace), Data Visualization (D3.js, Plotly)  
* **결과 저장 및 비교**  
  * 기술스택: DB (PostgreSQL, MongoDB), Vector DB (Pinecone, Weaviate, FAISS)

  ---

  ## **6\. 면접관 기능**

* **AI 질문 개입 가능**  
  * 기술스택: Real-time Dashboard (React, Next.js), WebSocket  
* **다대일 면접 지원**  
  * 기술스택: Networking (WebRTC, Socket.IO), Multi-user Session Management

  ---

  ## **7\. 고급 기능**

* **감독관 페르소나 지정**  
  * 기술스택: Prompt Engineering, Persona-based LLM Fine-tuning  
* **외부 데이터 활용 (RAG)**  
  * 기술스택: Vector DB (Pinecone, Milvus), Retrieval-Augmented Generation (LangChain, LlamaIndex)

  * 기술스택: Conversation Log DB, LLM Summarization  
* **SWOT 분석**  
  * 기술스택: NLP 기반 분석, Visualization Tools
